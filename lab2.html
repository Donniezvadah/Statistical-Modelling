Lab 2 — Linear Mixed Models

Analysing haematocrit recovery after hip replacement surgery

📋 Lab Overview
  • Topic: Linear mixed models for longitudinal data
  • Data: Haematocrit measurements from hip replacement patients
  • Package: lme4
  • Skills: EDA, model building, REML vs ML, BLUPs, prediction for new subjects
  • Solutions: Provided below each exercise — attempt independently first!

The Hip Replacement Dataset

Hip replacement surgery involves substantial blood loss. After surgery, a patient's haematocrit (the proportion of blood volume occupied by red blood cells, expressed as a percentage) drops and then gradually recovers. We are interested in:

  • The typical recovery trajectory over three time points
  • Whether males and females recover differently
  • Whether age affects recovery
  • Between-subject variability in recovery

Dataset Structure

VariableTypeDescription
yContinuousHaematocrit (%)
timeFactor (3 levels)1 = pre-op, 2 = post-op day 1, 3 = post-op day 4
sexBinary (0/1)0 = male, 1 = female
ageContinuousAge in years at time of surgery
subjIntegerSubject identifier (30 patients)

The dataset has 88 observations from 30 subjects. Each subject has measurements at up to 3 time points (some may have missing post-operative measurements).

Exercise 1 — Exploratory Data Analysis

📝 Exercise 1

Before fitting any model, explore the data carefully.

  1. Plot individual haematocrit profiles over time, separately for males and females. Connect the three time-point observations for each subject with a line. What patterns do you notice?
  2. Based on the plot, argue whether a random intercept is needed (i.e., do subjects differ systematically in their baseline haematocrit?).
  3. Does the slope of the recovery trajectory vary across subjects? Would a random slope for time improve the model?
  4. Compute summary statistics (mean, SD) of haematocrit by time and sex. Is there a visible sex difference?
Solution to Exercise 1
R — Solution
# ── 1. Individual profiles ───────────────────────────────────────
ggplot(hip, aes(x = time, y = y, group = subj, colour = sex)) +
  geom_line(alpha = 0.6) +
  geom_point(size = 1.8) +
  facet_wrap(~sex) +
  scale_colour_manual(values = c("steelblue", "tomato")) +
  labs(
    title    = "Individual haematocrit profiles by sex",
    subtitle = "Time: 1 = pre-op, 2 = day 1 post-op, 3 = day 4 post-op",
    x = "Time", y = "Haematocrit (%)"
  ) +
  theme_minimal(base_size = 12)

# ── 2 & 3. Random intercept and slope diagnostics ────────────────
# Fit naive OLS ignoring subject
fit_ols <- lm(y ~ time + sex + age, data = hip)
hip$resid_ols <- residuals(fit_ols)

# Plot OLS residuals by subject
ggplot(hip, aes(x = time, y = resid_ols, group = subj, colour = subj)) +
  geom_line(alpha = 0.5) +
  guides(colour = "none") +
  labs(title = "OLS residuals: each line = one subject",
       subtitle = "Lines hugging one side → random intercept needed",
       x = "Time", y = "OLS residual") +
  theme_minimal()

# ── 4. Summary by time and sex ───────────────────────────────────
hip |>
  group_by(time, sex) |>
  summarise(
    n    = n(),
    mean = round(mean(y), 2),
    sd   = round(sd(y),   2),
    .groups = "drop"
  ) |>
  print()

What to look for in the plots:

  • Random intercept: If the profiles are roughly parallel (same shape, different heights), a random intercept captures the between-subject baseline differences. In hip data, subjects start at quite different haematocrit levels — a random intercept is clearly needed.
  • Random slope: If the lines cross frequently, subjects differ in their rate of change — a random slope for time would improve fit. In this dataset, there is modest crossing, suggesting a random slope may be marginally beneficial but is not essential.
  • Sex difference: Females typically have lower baseline haematocrit (around 37–38%) compared to males (around 42–43%). The difference tends to persist across all three time points.

Exercise 2 — Building the Linear Mixed Model

📝 Exercise 2

Fit a sequence of LMMs using lme4::lmer(). The general model is:

$$Y_{ij} = \mathbf{x}_{ij}^\top \boldsymbol{\beta} + \mathbf{z}_{ij}^\top \mathbf{b}_i + \varepsilon_{ij}$$

where $\mathbf{x}_{ij}$ contains fixed effects (time, sex, age, interactions) and $\mathbf{z}_{ij}$ specifies the random effects structure for subject $i$.

  1. Fit the following four models using REML:
    • M1: Fixed: time + sex + age; Random: intercept per subject
    • M2: Fixed: time * sex + age; Random: intercept per subject
    • M3: Fixed: time * sex + age; Random: intercept + slope for time
    • M4: Fixed: time * sex + age + time:age; Random: intercept per subject
  2. Compare all four models using AIC and BIC (refit with ML for the AIC/BIC comparison). Which model does each criterion prefer?
  3. For the best model, interpret each fixed-effect coefficient. What is the estimated sex difference in haematocrit? How does it change over time?
  4. Extract and report the estimated variance components $\hat{\sigma}_b^2$ and $\hat{\sigma}^2$. Compute the intraclass correlation coefficient (ICC).
Solution to Exercise 2
R — Solution
library(lme4)

# ── Fit four models (REML for estimation) ───────────────────────
M1 <- lmer(y ~ time + sex + age + (1 | subj),
           data = hip, REML = TRUE)

M2 <- lmer(y ~ time * sex + age + (1 | subj),
           data = hip, REML = TRUE)

M3 <- lmer(y ~ time * sex + age + (time | subj),
           data = hip, REML = TRUE)

M4 <- lmer(y ~ time * sex + age + time:age + (1 | subj),
           data = hip, REML = TRUE)

# ── Refit with ML for AIC/BIC comparison ────────────────────────
M1_ml <- update(M1, REML = FALSE)
M2_ml <- update(M2, REML = FALSE)
M3_ml <- update(M3, REML = FALSE)
M4_ml <- update(M4, REML = FALSE)

aic_bic <- data.frame(
  Model = c("M1","M2","M3","M4"),
  AIC   = round(c(AIC(M1_ml), AIC(M2_ml), AIC(M3_ml), AIC(M4_ml)), 2),
  BIC   = round(c(BIC(M1_ml), BIC(M2_ml), BIC(M3_ml), BIC(M4_ml)), 2)
)
print(aic_bic)

# ── Summary of best model (say M2) ──────────────────────────────
summary(M2)

# ── Variance components ─────────────────────────────────────────
vc <- as.data.frame(VarCorr(M2))
print(vc)

# ICC
sigma_b2 <- vc$vcov[vc$grp == "subj"]
sigma2   <- vc$vcov[vc$grp == "Residual"]
icc      <- sigma_b2 / (sigma_b2 + sigma2)
cat(sprintf("sigma_b^2 = %.3f,  sigma^2 = %.3f,  ICC = %.3f\n",
            sigma_b2, sigma2, icc))

Interpreting the fixed effects (M2 as example):

  • time2: The change in haematocrit from pre-op to day 1 post-op for a male patient. Typically strongly negative (surgery causes blood loss).
  • time3: The change from pre-op to day 4 post-op for males. Should be less negative than time2 (some recovery by day 4).
  • sexFemale: Baseline (pre-op) haematocrit difference between females and males. Typically negative (females have lower haematocrit on average).
  • time2:sexFemale, time3:sexFemale: Whether the drop and recovery pattern differs between sexes.
  • age: Effect of age on haematocrit. Older patients may have lower baseline levels.
  • ICC: Typically around 0.6–0.8, meaning 60–80% of total variation is between subjects. This confirms the random intercept is important.

Exercise 4 — Predicting Incomplete Profiles

📝 Exercise 4

A key strength of mixed models is the ability to predict an individual's complete trajectory even when their data are incomplete.

  1. Identify two subjects in the dataset who have only pre-operative measurements (time point 1 only), or simulate two such "new" patients with known age and sex but only a pre-op haematocrit value.
  2. For each patient, compute:
    • The population prediction (fixed effects only, $\hat{b}_i = 0$)
    • The individual prediction using the BLUP $\hat{b}_i$ estimated from their single pre-op observation
  3. Compare the two predictions. How much does knowing the pre-op value improve the post-op prediction? Express the difference in haematocrit units.
  4. Extension: Derive analytically the BLUP formula for a random-intercept-only model when only one observation $y_{i1}$ is available. Show that the BLUP shrinks the observed deviation toward zero.
Solution to Exercise 4
R — Parts 1–3
# ── Create two "new" patients with only pre-op observation ───────
new_patients <- data.frame(
  subj = factor(c("New1", "New2")),
  age  = c(65, 72),
  sex  = factor(c("Male", "Female")),
  time = factor(c(1, 1)),
  y    = c(44.5, 38.2)   # their observed pre-op haematocrit
)

# ── Population prediction (b_i = 0) ─────────────────────────────
pred_grid <- expand.grid(
  time = factor(1:3),
  stringsAsFactors = FALSE
)
for (i in 1:nrow(new_patients)) {
  pt_data <- new_patients[i, ]
  pred_all_times <- lapply(1:3, function(t) {
    data.frame(
      subj = pt_data$subj,
      age  = pt_data$age,
      sex  = pt_data$sex,
      time = factor(t)
    )
  }) |> do.call(rbind)

  pop_pred <- predict(best_model, newdata = pred_all_times, re.form = NA)
  cat(sprintf("\nPatient %s (age=%d, %s, pre-op y=%.1f)\n",
              as.character(pt_data$subj), pt_data$age,
              as.character(pt_data$sex), pt_data$y))
  cat("Population predictions:", round(pop_pred, 2), "\n")
}

# ── Individual prediction via BLUP ───────────────────────────────
# Manually compute BLUP for random-intercept model
# BLUP(b_i) = sigma_b^2 / (sigma_b^2 + sigma^2/n_i) * (y_bar_i - x_i'beta_hat)

vc         <- as.data.frame(VarCorr(best_model))
sigma_b2   <- vc$vcov[vc$grp == "subj"]
sigma2     <- vc$vcov[vc$grp == "Residual"]
shrinkage  <- sigma_b2 / (sigma_b2 + sigma2 / 1)   # n_i = 1 obs

for (i in 1:nrow(new_patients)) {
  pt <- new_patients[i, ]
  # Population-predicted pre-op value
  pop_preop <- predict(best_model,
                       newdata = data.frame(subj=factor("NewX"),
                                            age=pt$age, sex=pt$sex, time=factor(1)),
                       re.form = NA)
  deviation  <- pt$y - pop_preop
  blup_b     <- shrinkage * deviation

  cat(sprintf(
    "\nPatient %s: deviation=%.2f, shrinkage=%.3f, BLUP(b_i)=%.2f\n",
    as.character(pt$subj), deviation, shrinkage, blup_b
  ))
  cat(sprintf(
    "Population post-op day4 prediction: %.2f\n",
    predict(best_model,
            newdata=data.frame(subj=factor("NewX"),
                               age=pt$age, sex=pt$sex, time=factor(3)),
            re.form=NA)
  ))
  cat(sprintf(
    "Individual  post-op day4 prediction: %.2f  (adjusted by %.2f)\n",
    predict(best_model,
            newdata=data.frame(subj=factor("NewX"),
                               age=pt$age, sex=pt$sex, time=factor(3)),
            re.form=NA) + blup_b,
    blup_b
  ))
}

Part 4: Analytic BLUP derivation.

For a random-intercept model $Y_{ij} = \mu_{ij} + b_i + \varepsilon_{ij}$ with $b_i \sim \mathrm{N}(0, \sigma_b^2)$ and $\varepsilon_{ij} \sim \mathrm{N}(0, \sigma^2)$, suppose we observe only one value $y_{i1}$ for subject $i$. Define $e_{i1} = y_{i1} - \mu_{i1}$ (observed deviation from fixed-effect prediction).

The BLUP minimises $\mathbb{E}[(b_i - \hat{b}_i)^2]$ over all linear estimators of $b_i$. By direct calculation:

$$\hat{b}_i = \frac{\sigma_b^2}{\sigma_b^2 + \sigma^2} e_{i1} = \rho \cdot e_{i1}$$

where $\rho = \sigma_b^2 / (\sigma_b^2 + \sigma^2) \in (0,1)$ is the ICC.

  • If ICC $\approx 1$ (most variation is between subjects): $\hat{b}_i \approx e_{i1}$ — we trust the single observation heavily.
  • If ICC $\approx 0$ (most variation is within-subject noise): $\hat{b}_i \approx 0$ — the single observation is too noisy to update our estimate.

With multiple observations $n_i$, the formula generalises to:

$$\hat{b}_i = \frac{n_i \sigma_b^2}{n_i \sigma_b^2 + \sigma^2} \bar{e}_i$$

As $n_i \to \infty$, $\hat{b}_i \to \bar{e}_i$ (we trust the data completely); as $n_i \to 0$, $\hat{b}_i \to 0$ (we revert to the population mean). This is the shrinkage property of BLUPs.

Exercise 5 — Testing Fixed Effects

📝 Exercise 5
  1. Test whether the time × sex interaction is statistically significant using a likelihood ratio test. Remember: use ML (not REML) for comparing models with different fixed effects.
  2. Test whether age has a significant effect on haematocrit.
  3. Use the anova() function in lme4 and interpret the output. What are the degrees of freedom for each test? Why?
Solution to Exercise 5
R — Solution
# ── All models fitted with ML ────────────────────────────────────
# M2: time * sex + age + (1|subj)  [includes interaction]
# M1: time + sex + age + (1|subj)  [no interaction]
# M_noage: time * sex + (1|subj)   [no age]

M2_ml    <- lmer(y ~ time * sex + age + (1|subj), data = hip, REML = FALSE)
M1_ml    <- lmer(y ~ time + sex + age + (1|subj), data = hip, REML = FALSE)
M_noage  <- lmer(y ~ time * sex + (1|subj),       data = hip, REML = FALSE)

# ── Test 1: Is time × sex interaction significant? ───────────────
lrt_interaction <- anova(M1_ml, M2_ml)
print(lrt_interaction)
# H0: no interaction (M1) vs H1: interaction present (M2)
# df difference = 2 (two extra interaction terms: time2:sex, time3:sex)

# ── Test 2: Is age significant? ──────────────────────────────────
lrt_age <- anova(M_noage, M2_ml)
print(lrt_age)
# H0: no age effect (M_noage) vs H1: age matters (M2_ml)
# df difference = 1

# ── Interpretation ────────────────────────────────────────────────
cat("\n--- Interaction test ---\n")
cat("Chi-squared =", round(lrt_interaction$Chisq[2], 3),
    "df =", lrt_interaction$`Df`[2],
    "p =", round(lrt_interaction$`Pr(>Chisq)`[2], 4), "\n")

cat("\n--- Age test ---\n")
cat("Chi-squared =", round(lrt_age$Chisq[2], 3),
    "df =", lrt_age$`Df`[2],
    "p =", round(lrt_age$`Pr(>Chisq)`[2], 4), "\n")

Notes on degrees of freedom:

  • The interaction test has 2 df because adding time * sex introduces two extra parameters: time2:sexFemale and time3:sexFemale (time has 3 levels, so 2 dummy variables).
  • The age test has 1 df (one additional parameter).
  • The LRT statistic $2[\ell(\hat{\boldsymbol{\theta}}_1) - \ell(\hat{\boldsymbol{\theta}}_0)]$ follows $\chi^2_{\Delta p}$ asymptotically under $H_0$, where $\Delta p$ is the difference in the number of parameters.

Lab Summary

What we learned from Lab 2

  • Exploratory analysis is essential before modelling: individual profile plots reveal the need for random effects far more clearly than any formal test.
  • Use REML for estimation of variance components and fixed effects in the final model; use ML for model comparison (AIC/BIC/LRT) when fixed effects differ between models.
  • The ICC quantifies how much of the total variation is between subjects. A high ICC (e.g. 0.7) means a random intercept captures most of the clustering structure.
  • BLUPs shrink individual estimates toward the population mean. With only one observation, the shrinkage factor equals the ICC. With many observations, the BLUP converges to the OLS estimate.
  • Mixed models enable prediction for new subjects with partial data by combining population-level estimates with subject-specific BLUPs.
