Random Effects Models

When data are clustered — students nested in schools, measurements repeated on the same individual, patients within hospitals — ignoring the clustering structure leads to anti-conservative inference. Random effects models account for between-cluster variability by treating cluster-level parameters as draws from a distribution rather than fixed constants. This chapter builds the theory from the simple one-way model through the full Linear Mixed Model, covering variance components, REML, BLUPs, and practical implementation in R.

1. Fixed vs Random Effects

Fixed EffectsRandom Effects
Levels of interest are all levels in the populationLevels are a random sample from a large population of levels
Estimate a separate parameter for each levelEstimate the distribution (variance) of level-specific effects
Inference applies only to observed levelsInference generalises to unobserved levels from the same population
Treatment groups, sex, species (discrete, exhaustive)Subjects, schools, batches, hospitals (sample from a large pool)
$p$ parameters added for $p$ groupsOnly $1$ extra parameter (the variance $\sigma_b^2$)
The key question: are the levels a population sample?

If you want to predict for a new level not seen in the data (new school, new patient), that level is random. If you only care about the exact levels observed, use fixed effects.

2. One-Way Random Effects Model

2.1 Model Definition

Definition 2.1 — One-Way Random Effects Model
$$Y_{ij} = \mu + b_i + \varepsilon_{ij}, \quad i = 1,\ldots,m, \quad j = 1,\ldots,n_i,$$

where:

  • $\mu$ is the overall population mean (fixed).
  • $b_i \sim N(0, \sigma_b^2)$ iid — random cluster effects (school $i$'s deviation from $\mu$).
  • $\varepsilon_{ij} \sim N(0, \sigma^2)$ iid — within-cluster errors.
  • $b_i$ and $\varepsilon_{ij}$ are independent.

Total number of observations: $N = \sum_{i=1}^m n_i$. Two variance parameters: $\sigma_b^2$ (between-cluster) and $\sigma^2$ (within-cluster).

2.2 Marginal and Conditional Distributions

Theorem 2.1 — Marginal Distribution

Marginalising over $b_i$:

$$Y_{ij} \sim N(\mu,\; \sigma_b^2 + \sigma^2).$$

Within cluster $i$, observations are correlated:

$$\Cov(Y_{ij}, Y_{ij'}) = \sigma_b^2 \quad (j \neq j'), \qquad \Var(Y_{ij}) = \sigma_b^2 + \sigma^2.$$

Between clusters: $\Cov(Y_{ij}, Y_{i'j'}) = 0$ for $i\neq i'$.

Proof
Proof

$\E[Y_{ij}] = \mu + \E[b_i] + \E[\varepsilon_{ij}] = \mu$.

$\Var(Y_{ij}) = \Var(b_i) + \Var(\varepsilon_{ij}) = \sigma_b^2 + \sigma^2$.

$\Cov(Y_{ij}, Y_{ij'}) = \Cov(b_i+\varepsilon_{ij},\, b_i+\varepsilon_{ij'}) = \Var(b_i) = \sigma_b^2$ for $j\neq j'$. $\square$

2.3 Intraclass Correlation Coefficient (ICC)

Definition 2.2 — ICC
$$\rho = \frac{\sigma_b^2}{\sigma_b^2 + \sigma^2} = \Cor(Y_{ij}, Y_{ij'}) \in [0, 1).$$

The ICC measures the proportion of total variance attributable to between-cluster differences. $\rho$ close to 1 means observations within a cluster are nearly identical; $\rho$ near 0 means clustering is irrelevant and a simple linear model suffices.

Example — School exam scores

Suppose $\sigma_b^2 = 25$ (between-school variance) and $\sigma^2 = 75$ (within-school variance). Then $\rho = 25/100 = 0.25$: 25% of variation is school-level. Two students from the same school are correlated at 0.25. Using OLS and ignoring school membership would underestimate SEs by factor $\approx\sqrt{1+(n-1)\rho}$ per cluster.

3. Estimating Variance Components

3.1 ANOVA / Method of Moments Estimators

For balanced data ($n_i = n$), the ANOVA approach equates expected mean squares to their observed values.

SourcedfSSMS$\E[\text{MS}]$
Between clusters $m-1$ $n\sum_i(\bar Y_{i\cdot}-\bar Y_{\cdot\cdot})^2$ $\text{MS}_B$ $\sigma^2 + n\sigma_b^2$
Within clusters $N-m$ $\sum_i\sum_j(Y_{ij}-\bar Y_{i\cdot})^2$ $\text{MS}_W$ $\sigma^2$

Method of moments estimators:

$$\hat\sigma^2 = \text{MS}_W, \qquad \hat\sigma_b^2 = \frac{\text{MS}_B - \text{MS}_W}{n}.$$
Negative variance estimates

If $\text{MS}_B < \text{MS}_W$, the MOM estimate $\hat\sigma_b^2 < 0$, which is inadmissible. In practice, truncate at zero. This happens when there is less between-cluster variation than expected. REML estimates are constrained to be non-negative.

4. The General Linear Mixed Model

Definition 4.1 — Linear Mixed Model (LMM)
$$\bY = \bX\bbeta + \bZ\bb + \beps,$$

where:

  • $\bY \in \mathbb{R}^N$ — response vector.
  • $\bX \in \mathbb{R}^{N\times p}$ — design matrix for fixed effects $\bbeta$.
  • $\bZ \in \mathbb{R}^{N\times q}$ — design matrix for random effects $\bb$.
  • $\bb \sim N(\mathbf{0}, \bD)$ — random effects, covariance matrix $\bD$ (to be estimated).
  • $\beps \sim N(\mathbf{0}, \sigma^2\bI)$ — residual errors, independent of $\bb$.

4.1 Marginal Distribution of $\bY$

Theorem 4.1 — Marginal Distribution

Marginalising over $\bb$:

$$\bY \sim N\bigl(\bX\bbeta,\; \bV\bigr), \quad \bV = \bZ\bD\bZ^T + \sigma^2\bI.$$

The marginal covariance $\bV$ encodes both the random effects structure and residual error.

Proof
Proof

$\E[\bY] = \bX\bbeta + \bZ\E[\bb] = \bX\bbeta$.

$$\Var(\bY) = \bZ\Var(\bb)\bZ^T + \Var(\beps) = \bZ\bD\bZ^T + \sigma^2\bI = \bV. \quad\square$$

4.2 GLS Estimator of Fixed Effects

With known $\bV$, the Generalised Least Squares (GLS) estimator of $\bbeta$ is:

$$\hat\bbeta_{\text{GLS}} = (\bX^T\bV^{-1}\bX)^{-1}\bX^T\bV^{-1}\bY,$$

which is BLUE among all linear unbiased estimators (Gauss–Markov extended to non-identity covariance). In practice $\bV$ is unknown; we plug in $\hat\bV$ (estimated by REML/ML) to get FGLS.

5. REML — Restricted Maximum Likelihood

5.1 Why Not Plain ML?

The ML estimator of variance components ignores the degrees of freedom used to estimate $\bbeta$, leading to downward bias — analogous to dividing by $n$ instead of $n-1$ in the sample variance. With $p$ fixed-effect parameters, the bias factor is $(N-p)/N$.

5.2 The REML Log-Likelihood

Definition 5.1 — REML

REML maximises the likelihood of $N-p$ error contrasts $\mathbf{K}^T\bY$ where $\mathbf{K}^T\bX = \mathbf{0}$ (contrasts orthogonal to $\bX$). The REML log-likelihood is:

$$\ell_R(\bD, \sigma^2) = -\frac{1}{2}\log|\bV| - \frac{1}{2}\log|\bX^T\bV^{-1}\bX| - \frac{1}{2}(\bY-\bX\hat\bbeta)^T\bV^{-1}(\bY-\bX\hat\bbeta).$$

The second term $-\frac{1}{2}\log|\bX^T\bV^{-1}\bX|$ is the REML correction; it penalises for estimating $\bbeta$, recovering unbiasedness for variance components.

ML vs REML — when to use each
MLREML
Fixed effectsUse for LRT on fixed effectsCorrect for fixed effects
Random effectsBiased downwardLess biased; use for LRT on random effects
AIC/BIC comparisonCan compare models with different fixed effectsCannot compare models with different fixed effects

Rule: use REML for estimating variance components; refit with ML to compare models differing in fixed-effect structure.

6. Best Linear Unbiased Predictors (BLUPs)

Definition 6.1 — BLUP

The BLUP of the random effect $\bb$ is the conditional expectation given $\bY$:

$$\tilde\bb = \E[\bb \mid \bY] = \bD\bZ^T\bV^{-1}(\bY - \bX\hat\bbeta).$$

It minimises $\E[(\bb - \hat\bb)^T(\bb - \hat\bb)]$ among all linear predictors.

Derivation of the BLUP formula
Derivation

The joint distribution of $(\bb, \bY)$ is: $\bb \sim N(\mathbf{0},\bD)$ and $\bY\mid\bb \sim N(\bX\bbeta+\bZ\bb, \sigma^2\bI)$. By the multivariate normal conditional formula:

$$\bb \mid \bY \sim N\!\left(\bD\bZ^T\bV^{-1}(\bY-\bX\bbeta),\;\bD - \bD\bZ^T\bV^{-1}\bZ\bD\right).$$

Replacing unknown $\bbeta$ with $\hat\bbeta$ gives the BLUP. $\square$

6.1 Shrinkage Interpretation

For the one-way model, the BLUP of cluster $i$'s effect is:

$$\tilde b_i = \underbrace{\frac{n_i\sigma_b^2}{n_i\sigma_b^2 + \sigma^2}}_{=:\;\lambda_i\;\text{ (shrinkage factor)}}\cdot\underbrace{(\bar Y_{i\cdot} - \hat\mu)}_{\text{raw cluster deviation}}.$$
Why shrinkage is a feature, not a bug
  • The shrinkage factor $\lambda_i \in (0,1)$: BLUPs are always pulled toward zero (the population mean).
  • $\lambda_i \to 1$ (no shrinkage) when $n_i$ is large or $\sigma_b^2 \gg \sigma^2$ — the cluster has a lot of data or clusters are very different.
  • $\lambda_i \to 0$ (full shrinkage) when $n_i$ is small or $\sigma_b^2 \approx 0$ — sparse clusters borrow strength from the population.
  • BLUPs are empirical Bayes estimates: they shrink toward the prior mean $0$ with a strength determined by the data.

7. Henderson's Mixed Model Equations

Henderson (1950) showed that $(\hat\bbeta, \tilde\bb)$ solve the joint linear system:

$$\begin{pmatrix}\bX^T\bX & \bX^T\bZ \\ \bZ^T\bX & \bZ^T\bZ + \sigma^2\bD^{-1}\end{pmatrix}\begin{pmatrix}\hat\bbeta \\ \tilde\bb\end{pmatrix} = \begin{pmatrix}\bX^T\bY \\ \bZ^T\bY\end{pmatrix}.$$

This is the mixed-model analogue of the normal equations. Solving this system avoids inverting the $N\times N$ matrix $\bV$, which is crucial for large datasets.

8. Random Slopes — Varying-Coefficient Models

Definition 8.1 — Random Intercepts and Slopes
$$Y_{ij} = (\beta_0 + b_{0i}) + (\beta_1 + b_{1i})x_{ij} + \varepsilon_{ij},$$

where $(b_{0i}, b_{1i})^T \sim N(\mathbf{0}, \bD)$ with

$$\bD = \begin{pmatrix}\sigma_{b_0}^2 & \sigma_{b_0 b_1} \\ \sigma_{b_0 b_1} & \sigma_{b_1}^2\end{pmatrix}.$$

This allows each cluster to have its own intercept and its own slope for $x$, with the cluster-specific slopes drawn from a bivariate normal distribution.

The correlation $\rho_{01} = \sigma_{b_0b_1}/(\sigma_{b_0}\sigma_{b_1})$ between random intercepts and slopes is of substantive interest. E.g., in education: do schools with higher average performance also show larger benefits of tutoring?

9. Nested vs Crossed Random Effects

StructureDescriptionlme4 formula
Nested: pupils within classes within schools Each lower level belongs to exactly one higher level (1|school/class)
Crossed: students $\times$ items Every student attempts every item; both are random (1|student) + (1|item)
Partially crossed Not every combination observed (1|student) + (1|item) (same syntax)

10. Testing Random Effects

Testing $H_0\colon\sigma_b^2 = 0$ (no random effect needed) is non-standard because $\sigma_b^2 = 0$ is on the boundary of the parameter space. The usual $\chi^2_1$ null distribution for the LRT is anticonservative.

Theorem 10.1 — Boundary LRT

For $H_0\colon\sigma_b^2=0$, the LRT statistic has the asymptotic null distribution:

$$G^2 \sim \tfrac{1}{2}\chi^2_0 + \tfrac{1}{2}\chi^2_1,$$

where $\chi^2_0$ is a point mass at 0 with probability $\frac{1}{2}$. The $p$-value is $\frac{1}{2}\Pr(\chi^2_1 > G^2_{\text{obs}})$ — half the usual $p$-value.

## ── 7. Test for random slope: boundary-corrected LRT ─────────
m0 <- lmer(Reaction ~ Days + (1|Subject),
           data = sleepstudy, REML = TRUE)
m_slope <- lmer(Reaction ~ Days + (Days|Subject),
                data = sleepstudy, REML = TRUE)
# REML-based LRT (comparing random effect structure):
lrt_stat <- as.numeric(2*(logLik(m_slope) - logLik(m0)))
# Boundary correction: mixture chi-sq(0) + chi-sq(2) for adding
# 2 parameters (sigma_slope^2 and correlation)
# Conservative p-value:
pval <- 0.5*(1 - pchisq(lrt_stat, df = 1)) +
        0.5*(1 - pchisq(lrt_stat, df = 2))
cat("LRT stat =", round(lrt_stat,2), "; p-value =", round(pval,4))
# LRT stat = 42.04 ; p-value = 0.0000

# Simulate a one-way random effects setup
set.seed(42)
school_data <- data.frame(
  school   = rep(1:20, each = 10),
  score    = rnorm(200, mean = 60, sd = 10)

  rep(rnorm(20, 0, 5), each = 10)   # add school effects

Key Takeaways

Summary

  • Random effects model between-cluster variability with a variance parameter $\sigma_b^2$, far more parsimonious than separate fixed effects for each cluster.
  • The ICC $\rho = \sigma_b^2/(\sigma_b^2+\sigma^2)$ quantifies clustering strength and the inflation of Type I error if ignored.
  • REML corrects for the downward bias in ML estimates of variance components; use REML for variance estimates, ML for comparing fixed-effect structures.
  • BLUPs $\tilde b_i$ are shrunken versions of cluster deviations — shrinkage is strongest for small clusters and weak between-cluster signal.
  • Testing $\sigma_b^2=0$ uses a boundary mixture $\frac{1}{2}\chi^2_0 + \frac{1}{2}\chi^2_1$; the naive $\chi^2_1$ $p$-value is approximately twice too small.
  • Use lme4::lmer() in R; add lmerTest for Satterthwaite-approximate $t$-tests on fixed effects.
