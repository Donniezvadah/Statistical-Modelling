Linear Regression

Linear regression is the foundation of all statistical modelling. Every advanced model — GLMs, mixed models, nonlinear regression — is built on ideas first encountered here. This chapter develops the theory from scratch: from a geometric view of OLS to the Gauss–Markov theorem, full distributional inference, and rigorous diagnostics — with complete proofs at every step and annotated R output.

1. The Regression Problem

We observe $n$ pairs $(x_i, y_i)$ and want to understand how the conditional mean of $Y$ given $X = x$ depends on $x$. The simplest assumption is linearity:

$$\E[Y \mid X = x] = \beta_0 + \beta_1 x.$$

Individual observations deviate from this mean by a random error term:

Definition 1.1 — Simple Linear Regression Model
$$Y_i = \beta_0 + \beta_1 x_i + \varepsilon_i, \quad i = 1,\ldots,n,$$

with assumptions:

  • Linearity: $\E[Y_i \mid x_i] = \beta_0 + \beta_1 x_i$.
  • Independence: $\varepsilon_1, \ldots, \varepsilon_n$ are mutually independent.
  • Homoscedasticity: $\Var(\varepsilon_i) = \sigma^2 < \infty$ for all $i$.
  • Normality (for inference): $\varepsilon_i \sim N(0,\sigma^2)$.

The acronym LINE summarises these. OLS estimation only requires L, I, H. Normality is added for exact finite-sample inference.

2. OLS for Simple Linear Regression

2.1 Derivation of the OLS Estimators

Ordinary Least Squares finds $\hat\beta_0, \hat\beta_1$ that minimise the Residual Sum of Squares:

$$\text{RSS}(\beta_0,\beta_1) = \sum_{i=1}^n \bigl(y_i - \beta_0 - \beta_1 x_i\bigr)^2.$$
Theorem 2.1 — OLS Estimators
$$\hat\beta_1 = \frac{S_{xy}}{S_{xx}}, \qquad \hat\beta_0 = \bar y - \hat\beta_1\bar x,$$

where $\bar x = \frac{1}{n}\sum x_i$, $\bar y = \frac{1}{n}\sum y_i$, $S_{xx} = \sum(x_i-\bar x)^2$, $S_{xy} = \sum(x_i-\bar x)(y_i-\bar y)$.

Complete proof
Proof

Setting partial derivatives to zero:

$$\frac{\partial\,\text{RSS}}{\partial\beta_0} = -2\sum(y_i - \beta_0 - \beta_1 x_i) = 0 \;\Rightarrow\; n\hat\beta_0 + \hat\beta_1\sum x_i = \sum y_i. \tag{i}$$ $$\frac{\partial\,\text{RSS}}{\partial\beta_1} = -2\sum x_i(y_i - \beta_0 - \beta_1 x_i) = 0 \;\Rightarrow\; \hat\beta_0\sum x_i + \hat\beta_1\sum x_i^2 = \sum x_i y_i. \tag{ii}$$

From (i): $\hat\beta_0 = \bar y - \hat\beta_1\bar x$. Substitute into (ii):

$$(\bar y - \hat\beta_1\bar x)\sum x_i + \hat\beta_1\sum x_i^2 = \sum x_i y_i$$ $$\hat\beta_1\!\left(\sum x_i^2 - \frac{(\sum x_i)^2}{n}\right) = \sum x_i y_i - \frac{\sum x_i\sum y_i}{n}$$

The left parenthesis equals $S_{xx}$ and the right side equals $S_{xy}$, giving $\hat\beta_1 = S_{xy}/S_{xx}$. Since $\partial^2\text{RSS}/\partial\beta_0^2 = 2n > 0$ and the Hessian is positive definite, this is a global minimum. $\square$

2.2 Geometric Interpretation

OLS projects $\mathbf{y}$ orthogonally onto the column space of $\bX$. The fitted vector $\hat{\mathbf{y}}$ is the closest point (in Euclidean distance) to $\mathbf{y}$ within $\text{col}(\bX)$. The residual vector $\hat{\beps} = \mathbf{y} - \hat{\mathbf{y}}$ is perpendicular to every column of $\bX$:

$$\bX^T\hat{\beps} = \mathbf{0} \quad \Longleftrightarrow \quad \text{normal equations}.$$

2.3 Properties of the Simple OLS Estimators

Theorem 2.2 — Moments of $\hat\beta_1$ and $\hat\beta_0$

Under assumptions L, I, H:

$$\E[\hat\beta_1] = \beta_1, \quad \Var(\hat\beta_1) = \frac{\sigma^2}{S_{xx}},$$ $$\E[\hat\beta_0] = \beta_0, \quad \Var(\hat\beta_0) = \sigma^2\!\left(\frac{1}{n}+\frac{\bar x^2}{S_{xx}}\right),$$ $$\Cov(\hat\beta_0,\hat\beta_1) = -\frac{\sigma^2\bar x}{S_{xx}}.$$
Proof
Proof

Write $\hat\beta_1 = \sum c_i y_i$ with $c_i = (x_i-\bar x)/S_{xx}$. Note $\sum c_i = 0$ and $\sum c_i x_i = 1$. Then:

$$\E[\hat\beta_1] = \sum c_i(\beta_0 + \beta_1 x_i) = \beta_0\underbrace{\sum c_i}_{0}+\beta_1\underbrace{\sum c_i x_i}_{1} = \beta_1.$$ $$\Var(\hat\beta_1) = \sigma^2\sum c_i^2 = \frac{\sigma^2\sum(x_i-\bar x)^2}{S_{xx}^2} = \frac{\sigma^2}{S_{xx}}.$$

For $\hat\beta_0 = \bar Y - \hat\beta_1\bar x$, use $\Var(\bar Y)=\sigma^2/n$ and $\Cov(\bar Y, \hat\beta_1) = \sigma^2\sum c_i/n = 0$ to get the formula for $\Var(\hat\beta_0)$. $\square$

3. Multiple Regression — Matrix Treatment

3.1 The Design Matrix

Definition 3.1 — Multiple Linear Regression

With $p$ predictors and $n$ observations:

$$\bY = \bX\bbeta + \beps, \quad \beps \sim (\mathbf{0},\,\sigma^2\bI_n),$$

where $\bY \in \R^n$, $\bX \in \R^{n\times(p+1)}$ (first column of ones for intercept), $\bbeta = (\beta_0,\ldots,\beta_p)^T$.

$$\bX = \begin{pmatrix}1&x_{11}&\cdots&x_{1p}\\\vdots&\vdots&&\vdots\\1&x_{n1}&\cdots&x_{np}\end{pmatrix}$$

3.2 The OLS Estimator

Theorem 3.1 — Matrix OLS

If $\text{rank}(\bX) = p+1$, the unique minimiser of $\|\bY - \bX\bbeta\|^2$ is:

$$\hat\bbeta = (\bX^T\bX)^{-1}\bX^T\bY.$$

Fitted values: $\hat\bY = \bH\bY$, residuals: $\hat\beps = (\bI-\bH)\bY$, where the hat matrix is

$$\bH = \bX(\bX^T\bX)^{-1}\bX^T.$$
Proof via matrix calculus
Proof

Expand the objective:

$$\text{RSS}(\bbeta) = (\bY-\bX\bbeta)^T(\bY-\bX\bbeta) = \|\bY\|^2 - 2\bbeta^T\bX^T\bY + \bbeta^T\bX^T\bX\bbeta.$$

Differentiating with respect to $\bbeta$:

$$\frac{\partial\,\text{RSS}}{\partial\bbeta} = -2\bX^T\bY + 2\bX^T\bX\bbeta \stackrel{!}{=} \mathbf{0} \;\Rightarrow\; \bX^T\bX\hat\bbeta = \bX^T\bY \quad (\text{normal equations}).$$

Full column rank $\Rightarrow$ $\bX^T\bX$ is positive definite and invertible, so $\hat\bbeta = (\bX^T\bX)^{-1}\bX^T\bY$ is the unique solution. The Hessian $2\bX^T\bX\succ 0$ confirms a global minimum. $\square$

3.3 Properties of the Hat Matrix $\bH$

Key Properties
  • Symmetric: $\bH = \bH^T$.
  • Idempotent: $\bH^2 = \bH$ (orthogonal projection onto $\text{col}(\bX)$).
  • Trace: $\operatorname{tr}(\bH) = p+1$; $\operatorname{tr}(\bI-\bH) = n-p-1$.
  • Eigenvalues: only 0 or 1.
  • Perpendicularity: $\bH(\bI-\bH) = \mathbf{0}$, so $\hat\bY \perp \hat\beps$.
  • Residuals uncorrelated with fitted: $\Cov(\hat\bY, \hat\beps) = \sigma^2\bH(\bI-\bH) = \mathbf{0}$.

4. The Gauss–Markov Theorem

Theorem 4.1 — Gauss–Markov (BLUE)

Assume $\E[\beps]=\mathbf{0}$ and $\Var(\beps)=\sigma^2\bI$ (no normality needed). For any fixed $\mathbf{c}\in\R^{p+1}$, the OLS estimator $\mathbf{c}^T\hat\bbeta$ is the Best Linear Unbiased Estimator (BLUE) of $\mathbf{c}^T\bbeta$: among all linear unbiased estimators $\tilde\bbeta = \mathbf{A}\bY$,

$$\Var(\mathbf{c}^T\hat\bbeta) \leq \Var(\mathbf{c}^T\tilde\bbeta).$$
Complete proof
Proof

Let $\tilde\bbeta = \mathbf{A}\bY$ be linear and unbiased: $\E[\mathbf{A}\bY] = \mathbf{A}\bX\bbeta = \bbeta$ for all $\bbeta$, so $\mathbf{A}\bX = \bI$.

Write $\mathbf{A} = (\bX^T\bX)^{-1}\bX^T + \mathbf{D}$ where $\mathbf{D} = \mathbf{A} - (\bX^T\bX)^{-1}\bX^T$. From $\mathbf{A}\bX = \bI$ it follows that $\mathbf{D}\bX = \mathbf{0}$.

$$\Var(\tilde\bbeta) = \sigma^2\mathbf{A}\mathbf{A}^T = \sigma^2\!\left[(\bX^T\bX)^{-1}\bX^T\bX(\bX^T\bX)^{-1} + (\bX^T\bX)^{-1}\bX^T\mathbf{D}^T + \mathbf{D}\bX(\bX^T\bX)^{-1} + \mathbf{D}\mathbf{D}^T\right]$$

The cross terms vanish since $\mathbf{D}\bX = \mathbf{0}$:

$$\Var(\tilde\bbeta) = \sigma^2(\bX^T\bX)^{-1} + \sigma^2\mathbf{D}\mathbf{D}^T = \Var(\hat\bbeta) + \sigma^2\mathbf{D}\mathbf{D}^T.$$

Since $\mathbf{D}\mathbf{D}^T \succeq 0$ (positive semidefinite), we have $\mathbf{c}^T\Var(\tilde\bbeta)\mathbf{c} \geq \mathbf{c}^T\Var(\hat\bbeta)\mathbf{c}$ for all $\mathbf{c}$. Equality iff $\mathbf{D} = \mathbf{0}$, i.e., $\tilde\bbeta = \hat\bbeta$. $\square$

What Gauss–Markov does and does NOT say
  • It guarantees optimality only within the class of linear estimators. Non-linear estimators (e.g., ridge regression) can have lower MSE by accepting bias.
  • It does not say OLS is optimal among all unbiased estimators — under normality, the Cramér–Rao lower bound gives this stronger result.
  • It requires $\Var(\beps) = \sigma^2\bI$. If errors are correlated or heteroscedastic, GLS is BLUE.

5. Distributional Theory under Normality

Add the assumption $\beps \sim N(\mathbf{0},\sigma^2\bI)$.

Theorem 5.1 — Distributions of Key Quantities
$$\hat\bbeta \sim N\!\bigl(\bbeta,\;\sigma^2(\bX^T\bX)^{-1}\bigr)$$ $$\hat\bY \sim N\!\bigl(\bX\bbeta,\;\sigma^2\bH\bigr)$$ $$\hat\beps \sim N\!\bigl(\mathbf{0},\;\sigma^2(\bI-\bH)\bigr)$$

Furthermore, $\hat\bbeta$ and $\hat\beps$ are independent, and:

$$\frac{\text{RSS}}{\sigma^2} = \frac{\hat\beps^T\hat\beps}{\sigma^2} \sim \chi^2_{n-p-1}.$$

The unbiased estimator of $\sigma^2$ is $\hat\sigma^2 = \text{RSS}/(n-p-1)$.

Proof of independence and chi-squared result
Proof

Since $\hat\bbeta = (\bX^T\bX)^{-1}\bX^T\bY$ and $\hat\beps = (\bI-\bH)\bY$ are both linear functions of $\bY \sim N(\bX\bbeta, \sigma^2\bI)$, they are jointly normal. Their covariance is:

$$\Cov(\hat\bbeta, \hat\beps) = (\bX^T\bX)^{-1}\bX^T \cdot \sigma^2\bI \cdot (\bI-\bH)^T = \sigma^2(\bX^T\bX)^{-1}\bX^T(\bI-\bH) = \mathbf{0}$$

since $\bX^T(\bI-\bH) = \bX^T - \bX^T\bH = \bX^T - \bX^T = \mathbf{0}$ (using $\bX^T\bH = \bX^T$). Zero covariance + joint normality $\Rightarrow$ independence.

For the chi-squared result: $\hat\beps = (\bI-\bH)\bY = (\bI-\bH)\beps$ (since $(\bI-\bH)\bX = \mathbf{0}$). The matrix $(\bI-\bH)/\sigma$ is idempotent with rank $n-p-1$, so $\hat\beps^T\hat\beps/\sigma^2 \sim \chi^2_{n-p-1}$. $\square$

6. Inference: Tests and Confidence Intervals

6.1 $t$-Test for a Single Coefficient

To test $H_0\colon \beta_j = 0$:

Theorem 6.1 — $t$-Test
$$T_j = \frac{\hat\beta_j}{\hat\sigma\sqrt{[(\bX^T\bX)^{-1}]_{jj}}} \sim t_{n-p-1} \quad \text{under } H_0.$$

The $95\%$ confidence interval for $\beta_j$ is:

$$\hat\beta_j \pm t_{n-p-1,\,0.025}\cdot\hat\sigma\sqrt{[(\bX^T\bX)^{-1}]_{jj}}.$$
Why $t_{n-p-1}$?
Derivation

$\hat\beta_j \sim N(\beta_j, \sigma^2 v_{jj})$ where $v_{jj} = [(\bX^T\bX)^{-1}]_{jj}$. Under $H_0$: $Z = \hat\beta_j/(\sigma\sqrt{v_{jj}}) \sim N(0,1)$.

Since $\text{RSS}/\sigma^2 \sim \chi^2_{n-p-1}$ independently of $\hat\bbeta$, and $\hat\sigma^2 = \text{RSS}/(n-p-1)$:

$$T_j = \frac{Z}{\sqrt{\chi^2_{n-p-1}/(n-p-1)}} = \frac{\hat\beta_j/(\sigma\sqrt{v_{jj}})}{\hat\sigma/\sigma} \sim t_{n-p-1}. \quad\square$$

6.2 $F$-Test — Overall Significance

The ANOVA decomposition:

$$\underbrace{\|\bY-\bar Y\mathbf{1}\|^2}_{\text{SST} = S_{yy}} = \underbrace{\|\hat\bY-\bar Y\mathbf{1}\|^2}_{\text{SSR}} + \underbrace{\|\bY-\hat\bY\|^2}_{\text{RSS}}$$
Theorem 6.2 — $F$-Test

Testing $H_0\colon \beta_1=\cdots=\beta_p=0$:

$$F = \frac{\text{SSR}/p}{\text{RSS}/(n-p-1)} \sim F_{p,\,n-p-1} \quad \text{under }H_0.$$

More generally, to test a reduced model (with $q$ constraints) against the full model:

$$F = \frac{(\text{RSS}_{\text{red}} - \text{RSS}_{\text{full}})/q}{\text{RSS}_{\text{full}}/(n-p-1)} \sim F_{q,\,n-p-1} \quad \text{under }H_0.$$
Proof sketch
Proof sketch

Under normality: $\text{SSR}/\sigma^2 \sim \chi^2_p$ (since the projection onto a $p$-dimensional space applied to a normal vector gives a chi-squared with $p$ df). $\text{RSS}/\sigma^2 \sim \chi^2_{n-p-1}$ independently (Cochran's theorem). The ratio of two independent chi-squared variables, each divided by their degrees of freedom, is $F$-distributed by definition. $\square$

6.3 Confidence and Prediction Intervals

For a new covariate vector $\bx_0 = (1, x_{0,1},\ldots,x_{0,p})^T$, let $\hat\mu_0 = \bx_0^T\hat\bbeta$.

IntervalFormulaTarget
Mean response CI $\hat\mu_0 \pm t_{n-p-1,\alpha/2}\;\hat\sigma\sqrt{\bx_0^T(\bX^T\bX)^{-1}\bx_0}$ $\E[Y_0|\bx_0] = \bx_0^T\bbeta$
Future observation PI $\hat y_0 \pm t_{n-p-1,\alpha/2}\;\hat\sigma\sqrt{1+\bx_0^T(\bX^T\bX)^{-1}\bx_0}$ New $Y_0 = \bx_0^T\bbeta + \varepsilon_0$
Why the PI is always wider than the CI

The extra term $+1$ inside the square root accounts for the variance of a new, unseen error $\varepsilon_0 \sim N(0,\sigma^2)$. Even with an infinite sample ($n\to\infty$, so $(\bX^T\bX)^{-1}\to\mathbf{0}$), the PI width $\to 2z_{\alpha/2}\sigma$ — you can never perfectly predict a single future value, only narrow the uncertainty around its mean.

7. Coefficient of Determination $R^2$

$$R^2 = \frac{\text{SSR}}{\text{SST}} = 1 - \frac{\text{RSS}}{\text{SST}},$$

the proportion of total variance explained by the model. $R^2$ equals the squared sample correlation between $y_i$ and $\hat y_i$.

Definition 7.1 — Adjusted $R^2$
$$R^2_{\text{adj}} = 1 - \frac{\text{RSS}/(n-p-1)}{\text{SST}/(n-1)} = 1 - (1-R^2)\frac{n-1}{n-p-1}.$$

Adding a predictor always increases $R^2$ but increases $R^2_{\text{adj}}$ only if the predictor's $F$-contribution exceeds 1 (i.e., it explains more than a random noise variable would).

Do not use $R^2$ for model selection

$R^2$ is a measure of fit, not predictive accuracy. A model with 50 noise predictors will have higher $R^2$ than a correctly-specified model with 3 predictors. Use AIC, BIC, or cross-validated prediction error for selection (Chapter 1).

8.3 Leverage and Influence Measures

Definition 8.1 — Leverage
$$h_{ii} = \bx_i^T(\bX^T\bX)^{-1}\bx_i = [\bH]_{ii}.$$

Properties: $1/n \leq h_{ii} \leq 1$; $\sum h_{ii} = p+1$; high-leverage threshold: $h_{ii} > 2(p+1)/n$.

High leverage means observation $i$ lies far from the centroid of $\bX$. It can influence $\hat\bbeta$ greatly but need not — only when combined with a large residual.

Definition 8.2 — Cook's Distance
$$D_i = \frac{(\hat\bbeta_{(-i)}-\hat\bbeta)^T\bX^T\bX(\hat\bbeta_{(-i)}-\hat\bbeta)}{(p+1)\hat\sigma^2} = \frac{r_i^2}{p+1}\cdot\frac{h_{ii}}{1-h_{ii}}.$$

$D_i > 1$ (conservative) or $D_i > 4/n$ (strict) flags influential observations.

The shortcut formula shows Cook's $D$ is large when both the standardised residual $|r_i|$ and the leverage $h_{ii}/(1-h_{ii})$ are large.

8.4 Variance Inflation Factor (VIF)

When predictors are correlated, $\bX^T\bX$ becomes near-singular, inflating all standard errors.

Definition 8.3 — VIF

Regress $x_j$ on all other predictors; let $R_j^2$ be the $R^2$. Then:

$$\text{VIF}_j = \frac{1}{1-R_j^2}, \qquad \Var(\hat\beta_j) = \frac{\sigma^2}{S_{x_jx_j}}\cdot\text{VIF}_j.$$

$\text{VIF}_j = 1$: no collinearity. $\text{VIF}_j > 10$: severe problem.

Key Takeaways

Summary

  • OLS minimises RSS; the solution $\hat\bbeta = (\bX^T\bX)^{-1}\bX^T\bY$ is found from the normal equations.
  • Gauss–Markov: OLS is BLUE under linearity, independence, and homoscedasticity — no normality needed.
  • Under normality: $\hat\bbeta \sim N(\bbeta, \sigma^2(\bX^T\bX)^{-1})$; exact $t$- and $F$-tests follow.
  • The hat matrix $\bH$ controls leverage; $h_{ii}$ and Cook's $D_i$ identify high-leverage and influential points.
  • Always run the four base-R diagnostic plots. No regression is complete without them.
  • $R^2$ measures fit, not model quality. Use AIC/BIC for model selection.
