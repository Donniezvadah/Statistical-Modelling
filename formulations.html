The Foundation Models

📐
Linear Model
LM
Foundation
Model
$$Y_i = \mathbf{x}_i^\top \boldsymbol{\beta} + \varepsilon_i, \qquad \varepsilon_i \overset{\mathrm{iid}}{\sim} \mathrm{N}(0,\sigma^2)$$
Equivalent form
$\mathbf{Y} \sim \mathrm{N}(\mathbf{X}\boldsymbol{\beta},\, \sigma^2 \mathbf{I}_n)$
Link function
Identity: $g(\mu) = \mu$
Estimation
OLS / MLE: $\hat{\boldsymbol{\beta}} = (\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{X}^\top\mathbf{y}$
Residual var.
$\hat{\sigma}^2 = \|\mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}}\|^2 / (n-p)$
Inference
Exact $t$ and $F$ tests; confidence intervals $\hat{\beta}_j \pm t_{n-p,\alpha/2}\,\mathrm{se}(\hat{\beta}_j)$
Assumptions
Linearity, homoscedasticity, normality, independence of errors
Diagnostics
Residuals vs fitted; Q–Q plot; Scale-location; Cook's distance; plot(lm(...))
R function
lm(y ~ x1 + x2, data = df)
Complexity
 1 / 5
Normal response Continuous X Closed-form MLE Exact inference
🔗
Generalised Linear Model
GLM
Foundation
Three components
  1. Random: $Y_i \mid \mathbf{x}_i \sim \mathrm{EF}(\mu_i,\phi)$ — exponential family
  2. Systematic: $\eta_i = \mathbf{x}_i^\top\boldsymbol{\beta}$ — linear predictor
  3. Link: $g(\mu_i) = \eta_i$ — connects mean to predictor
EF density
$f(y_i;\theta_i,\phi) = \exp\!\left\{\frac{y_i\theta_i - b(\theta_i)}{a(\phi)} + c(y_i,\phi)\right\}$
Mean & variance
$\mu_i = b'(\theta_i)$;   $\mathrm{Var}(Y_i) = a(\phi)\,b''(\theta_i)$
Common cases
DistributionLink$V(\mu)$Use
NormalIdentity$1$Continuous
BinomialLogit$\mu(1{-}\mu)$Binary / proportion
PoissonLog$\mu$Counts
GammaLog / Inverse$\mu^2$Positive continuous
Inv-Gaussian$1/\mu^2$$\mu^3$Positive, right-skewed
Estimation
Iteratively Reweighted Least Squares (IWLS); equivalent to Newton–Raphson on log-likelihood
Score equation
$\mathbf{X}^\top \mathbf{W}^{1/2}\mathbf{z} = \mathbf{0}$ where $z_i = \eta_i + (y_i-\mu_i)g'(\mu_i)$, $w_i = [V(\mu_i)g'(\mu_i)^2]^{-1}$
Deviance
$D = 2[\ell(\text{saturated}) - \ell(\hat{\boldsymbol{\beta}})]$; under correct model $D \approx \chi^2_{n-p}$
R function
glm(y ~ x, family = binomial(link="logit"))
Complexity
 2 / 5
Exponential family Link function IWLS Deviance tests Chapter 2 baseline

Overdispersion Models

〜
Quasi-Likelihood Model
Quasi-GLM
Chapter 2 — §2.2
Specification
Only first two moments: $\E[Y_i|\mathbf{x}_i] = \mu_i$,  $\Var(Y_i|\mathbf{x}_i) = \phi\, V(\mu_i)/m_i$
Quasi-score
$$U(\boldsymbol{\beta}) = \sum_{i=1}^n \frac{(y_i - \mu_i)}{\phi V(\mu_i)/m_i} \cdot \frac{\partial\mu_i}{\partial\boldsymbol{\beta}} = \mathbf{0}$$
Dispersion est.
$\hat{\phi} = \frac{1}{n-p}\sum_{i=1}^n \frac{(y_i-\hat\mu_i)^2}{V(\hat\mu_i)/m_i}$   (Pearson chi-squared / df)
SE adjustment
All standard errors multiplied by $\sqrt{\hat\phi}$; $t$-tests use $t_{n-p}$ distribution
When to use
Residual deviance / df $> 1.5$; model otherwise correct; no fully specified parametric alternative available
Limitation
No log-likelihood → AIC undefined; use quasi-AIC or QBIC for model selection
R function
glm(y ~ x, family = quasipoisson) or quasibinomial
Complexity
 2 / 5
Mean-variance only No full likelihood Robust SE Overdispersion
📊
Negative Binomial Regression
NB-GLM
Chapter 2 — §2.2
Latent structure
$U_i \sim \mathrm{Gamma}(\nu, \nu/\mu_i)$,  $Y_i|U_i \sim \mathrm{Poisson}(U_i)$
Marginal pmf
$P(Y_i=y) = \binom{y+\nu-1}{y}\!\left(\frac{\nu}{\nu+\mu_i}\right)^\nu\!\left(\frac{\mu_i}{\nu+\mu_i}\right)^y$
Mean & variance
$\E[Y_i] = \mu_i$;   $\Var(Y_i) = \mu_i + \mu_i^2/\nu$
Link function
Log: $\log\mu_i = \mathbf{x}_i^\top\boldsymbol{\beta}$
Overdispersion
$\phi = 1 + \mu_i/\nu$; as $\nu\to\infty$, Poisson recovered
Estimation
Profile MLE or EM (treat $U_i$ as latent; see Chapter 4, Example 4.3)
R function
MASS::glm.nb(y ~ x, data = df)
Complexity
 3 / 5
Count data Overdispersion Poisson–Gamma mixture Full likelihood

Mixed Models

🔀
Linear Mixed Model
LMM
Chapter 2 — §2.4
Model
$$\mathbf{Y}_i = \mathbf{X}_i\boldsymbol{\beta} + \mathbf{Z}_i\mathbf{b}_i + \boldsymbol{\varepsilon}_i$$
Distributions
$\mathbf{b}_i \sim \N(\mathbf{0}, \boldsymbol{\Sigma}_b)$;  $\boldsymbol{\varepsilon}_i \sim \N(\mathbf{0}, \sigma^2\mathbf{I}_{n_i})$;  $\mathbf{b}_i \perp \boldsymbol{\varepsilon}_i$
Marginal dist.
$\mathbf{Y}_i \sim \N\!\left(\mathbf{X}_i\boldsymbol{\beta},\; \underbrace{\mathbf{Z}_i\boldsymbol{\Sigma}_b\mathbf{Z}_i^\top + \sigma^2\mathbf{I}}_{\boldsymbol{\Sigma}_i}\right)$
Fixed effects
$\boldsymbol{\beta}$ — population-level parameters (GLS estimator)
Random effects
$\mathbf{b}_i$ — cluster-specific deviations (predicted by BLUP)
BLUP
$\hat{\mathbf{b}}_i = \boldsymbol{\Sigma}_b\mathbf{Z}_i^\top\boldsymbol{\Sigma}_i^{-1}(\mathbf{y}_i - \mathbf{X}_i\hat{\boldsymbol{\beta}})$
ICC
$\rho = \sigma_b^2/(\sigma_b^2 + \sigma^2)$ (random intercept model)
Estimation
REML for variance components; ML for fixed-effect model comparison
Model comparison
Use ML + AIC/BIC/LRT for fixed effects; use REML + LRT for random effects
R function
lme4::lmer(y ~ x + (1|subj))
Complexity
 3 / 5
Clustered data REML BLUP / shrinkage Hierarchical Normal response
🔗🔀
Generalised Linear Mixed Model
GLMM
Chapter 2 — §2.5
Conditional model
$$Y_{ij}|\mathbf{b}_i \sim \mathrm{EF}(\mu_{ij},\phi), \quad g(\mu_{ij}) = \mathbf{x}_{ij}^\top\boldsymbol{\beta} + \mathbf{z}_{ij}^\top\mathbf{b}_i$$
Random effects
$\mathbf{b}_i \overset{\mathrm{iid}}{\sim} \N(\mathbf{0}, \boldsymbol{\Sigma}_b)$
Marginal likelihood
$\ell = \sum_i \log\int f(\mathbf{y}_i|\mathbf{b}_i)\,\phi(\mathbf{b}_i;\boldsymbol{\Sigma}_b)\,d\mathbf{b}_i$  — no closed form for non-normal responses
Approximations
  • GHQ: Gauss–Hermite quadrature with $Q$ nodes (accurate, slow for high-dim $\mathbf{b}$)
  • Laplace: 2nd-order Taylor at posterior mode (fast, slightly less accurate)
  • PQL: Penalised quasi-likelihood (fast but biased — avoid for binary data)
Fixed vs. marginal
$\boldsymbol{\beta}$ = subject-specific effect (not population-averaged; different from GEE)
R function
lme4::glmer(y ~ x + (1|id), family=binomial, nAGQ=25)
Complexity
 4 / 5
Non-normal response Clustered data Quadrature / Laplace Subject-specific effects
⟺
Generalised Estimating Equations
GEE
Chapter 2 — §2.3
Model
$\E[Y_{ij}] = \mu_{ij}$,  $g(\mu_{ij}) = \mathbf{x}_{ij}^\top\boldsymbol{\beta}$,  $\Var(Y_{ij}) = \phi V(\mu_{ij})/w_{ij}$
Working corr.
$\mathrm{Cor}(Y_{ij},Y_{ik}) = R_{jk}(\boldsymbol{\alpha})$ — specified but need not be correct
Estimating eqs
$\sum_i \mathbf{D}_i^\top \mathbf{V}_i^{-1}(\mathbf{y}_i - \boldsymbol{\mu}_i) = \mathbf{0}$
Sandwich SE
$\hat{\mathbf{V}}(\hat{\boldsymbol{\beta}}) = \mathbf{A}^{-1}\mathbf{B}\mathbf{A}^{-1}$ — valid even if $\mathbf{R}$ is wrong
Interpretation
Population-averaged effect (different from GLMM subject-specific)
Working correlations
Independence · Exchangeable · AR(1) · Unstructured
R function
gee::gee() or geepack::geeglm()
Complexity
 3 / 5
Population-averaged Robust inference Any response type Longitudinal

Nonlinear Models

〰️
Nonlinear Least Squares
NLS
Chapter 3 — §3.2
Model
$$Y_i = \eta(x_i,\boldsymbol{\beta}) + \varepsilon_i, \quad \varepsilon_i \overset{\mathrm{iid}}{\sim} \N(0,\sigma^2)$$
Estimation
Minimise $S(\boldsymbol{\beta}) = \sum(y_i - \eta(x_i,\boldsymbol{\beta}))^2$ — iterative (Gauss–Newton)
Gauss–Newton step
$\boldsymbol{\beta}^{(t+1)} = \boldsymbol{\beta}^{(t)} + (\mathbf{J}^{(t)\top}\mathbf{J}^{(t)})^{-1}\mathbf{J}^{(t)\top}\mathbf{r}^{(t)}$
Jacobian
$J_{ij} = \partial\eta(x_i,\boldsymbol{\beta})/\partial\beta_j$ — $n\times p$ matrix of partial derivatives
Asymptotic cov.
$\widehat{\Var}(\hat{\boldsymbol{\beta}}) = \hat\sigma^2(\hat{\mathbf{J}}^\top\hat{\mathbf{J}})^{-1}$
Confidence int.
Profile likelihood CIs preferred over Wald CIs for skewed sampling distributions
Key caution
Requires good starting values; may converge to local minimum; always check with multiple starts
R function
nls(y ~ eta(x, b0, b1), start=list(b0=..., b1=...))
Complexity
 3 / 5
Mechanistic models Physical parameters Gauss–Newton Profile likelihood CI
〰️🔀
Nonlinear Mixed Effects Model
NLME
Chapter 3 — §3.3
Model
$$Y_{ij} = \eta(\boldsymbol{\beta} + \mathbf{A}\mathbf{b}_i,\, x_{ij}) + \varepsilon_{ij}$$
Random effects
$\mathbf{b}_i \sim \N(\mathbf{0}, \boldsymbol{\Sigma}_b)$;  $\mathbf{A}$ selects which params vary by cluster
Individual params
$\boldsymbol{\beta}_i = \boldsymbol{\beta} + \mathbf{A}\mathbf{b}_i$  — subject-specific parameter vector
Log-scale trick
Work with $\gamma_k = \log\beta_k$ to ensure positivity; $\beta_{ij} = \exp(\gamma_k + b_{ik})$ is log-normal
Predictions
  • Population: $\hat Y^{\mathrm{pop}} = \eta(\hat{\boldsymbol{\beta}}, x)$ — for a new subject
  • Individual: $\hat Y^{\mathrm{ind}} = \eta(\hat{\boldsymbol{\beta}} + \hat{\mathbf{b}}_i, x)$ — for observed subject
Estimation
Laplace approx. to marginal likelihood (nlme/nlmer); REML for variance components
R function
nlme::nlme() or lme4::nlmer()
Complexity
 4 / 5
Pharmacokinetics Repeated measures Physical parameters Log-normal random effects

Latent Variable Models

🎲
Finite Mixture Model
FMM
Chapter 4 — §4.3
Density
$$f(y;\boldsymbol{\pi},\boldsymbol{\theta}) = \sum_{k=1}^K \pi_k\, f_k(y;\theta_k)$$
Latent variable
$U_i \in \{1,\ldots,K\}$ with $P(U_i=k)=\pi_k$ — component membership
Mixing weights
$\pi_k \geq 0$, $\sum_k \pi_k = 1$
Gaussian case
$f_k = \N(\mu_k,\sigma_k^2)$; parameters $\boldsymbol{\theta} = \{(\mu_k,\sigma_k^2)\}_{k=1}^K$
Non-regularity
Label switching ($K!$ equivalent maxima); unbounded likelihood (degenerate components); set $\sigma_k^2 \geq \sigma^2_{\min}$
Estimation
EM algorithm (Section 4.5); multiple random starts essential
Select $K$
BIC (not LRT — regularity fails); bootstrap LRT; ICL criterion
R function
mclust::Mclust(y, G=1:10)
Complexity
 3 / 5
Clustering EM algorithm Multimodal data BIC for K Non-regular MLE

Information Criterion Formulas

📐 All information criteria at a glance
CriterionFormulaPenalty / paramGoal
AIC $2p - 2\ell(\hat{\boldsymbol{\theta}})$ $2$ Predictive accuracy (KL)
AIC$_c$ $\mathrm{AIC} + \frac{2p(p+1)}{n-p-1}$ $2 + \frac{2(p+1)}{n-p-1}$ AIC for small $n$
BIC $p\log n - 2\ell(\hat{\boldsymbol{\theta}})$ $\log n$ Bayesian marginal likelihood
TIC $2\,\mathrm{tr}(\hat J^{-1}\hat K) - 2\ell(\hat{\boldsymbol{\theta}})$ $2\,\mathrm{tr}(\hat J^{-1}\hat K)/p \approx 2$ AIC under misspecification
$C_p$ $\mathrm{RSS}_p/\hat\sigma^2 - n + 2p$ $2$ Prediction error (linear models)
LOO-CV $\frac{1}{n}\sum_i\!\left(\frac{e_i}{1-h_{ii}}\right)^2$ — Direct prediction error estimate

All criteria are to be minimised. For model comparison, only differences matter; the scale is the same as $-2\log$-likelihood.

Model Decision Guide

🗺️ Which model should I use?
Start here: What is your response variable?
│
├─ Continuous (normal errors likely)
│   ├─ Independent observations?
│   │   ├─ Linear relationship?    → LM        (lm)
│   │   └─ Nonlinear relationship? → NLS       (nls)
│   └─ Clustered / repeated?
│       ├─ Linear relationship?    → LMM       (lmer)
│       └─ Nonlinear relationship? → NLME      (nlme / nlmer)
│
├─ Binary (0/1)
│   ├─ Independent observations?
│   │   └─ Logistic / probit       → GLM       (glm, family=binomial)
│   └─ Clustered?
│       ├─ Subject-specific focus? → GLMM      (glmer, nAGQ ≥ 10)
│       └─ Population-avg focus?   → GEE       (geeglm)
│
├─ Count (0, 1, 2, ...)
│   ├─ Var ≈ Mean (no overdispersion)?
│   │   └─ Poisson GLM             → GLM       (glm, family=poisson)
│   ├─ Var >> Mean (overdispersion)?
│   │   ├─ Want full likelihood?   → Neg-Bin   (glm.nb)
│   │   └─ Just need correct SE?   → Quasi     (quasipoisson)
│   └─ Clustered counts?
│       └─ GLMM with Poisson       → GLMM      (glmer, family=poisson)
│
└─ Multimodal / grouped unknown membership
    └─ Mixture model               → FMM + EM  (Mclust)
