Generalised Linear Models

Generalised Linear Models (GLMs) unify regression, logistic regression, Poisson regression, gamma regression, and many more under one elegant framework. The key insight is that all these models share the same estimation algorithm (IRLS) and the same inferential machinery (deviance, LRT, Wald tests) — only the distribution family and link function differ. This chapter builds the entire GLM framework from the exponential family, derives the score equations and Fisher information, explains deviance-based inference, and gives fully worked examples for the most important special cases.

1. The Exponential Family

Definition 1.1 — Exponential Family (Canonical Form)

A distribution belongs to the exponential family if its density/pmf can be written as:

$$f(y;\,\theta,\phi) = \exp\!\left\{\frac{y\theta - b(\theta)}{a(\phi)} + c(y,\phi)\right\},$$

where:

  • $\theta$ — canonical (natural) parameter, depends on the mean $\mu$.
  • $\phi > 0$ — dispersion parameter (fixed or estimated separately).
  • $b(\theta)$ — cumulant function: generates moments via derivatives.
  • $a(\phi)$ — dispersion function, typically $a(\phi) = \phi/w_i$ for observation weight $w_i$.
  • $c(y,\phi)$ — normalising constant (not a function of $\theta$).

1.1 Moments from the Cumulant Function

Theorem 1.1 — Mean and Variance from $b(\theta)$
$$\E[Y] = \mu = b'(\theta), \qquad \Var(Y) = a(\phi)\,b''(\theta) = a(\phi)\,V(\mu),$$

where $V(\mu) = b''(\theta)$ is the variance function — the variance as a function of the mean, a key characteristic of each family.

Proof via moment generating property
Proof

The log-likelihood for one observation is $\ell(\theta) = [y\theta - b(\theta)]/a(\phi) + c(y,\phi)$. Since $\int f\,dy = 1$:

$$\frac{d}{d\theta}\int f\,dy = 0 \;\Rightarrow\; \int \frac{y - b'(\theta)}{a(\phi)} f\,dy = 0 \;\Rightarrow\; \E[Y] = b'(\theta).$$

Differentiating again: $\E\!\left[\left(\frac{Y-b'(\theta)}{a(\phi)}\right)^{\!2}\right] = \frac{b''(\theta)}{a(\phi)}$, giving $\Var(Y) = a(\phi)b''(\theta)$. $\square$

2. The Three Components of a GLM

Definition 2.1 — Generalised Linear Model

A GLM consists of three components:

  1. Random component: $Y_i \mid \bx_i$ are independent, each from the same exponential family with mean $\mu_i$.
  2. Systematic component: the linear predictor $\eta_i = \bx_i^T\bbeta = \beta_0 + \beta_1x_{i1}+\cdots+\beta_px_{ip}$.
  3. Link function: a monotone differentiable function $g$ such that $g(\mu_i) = \eta_i$, i.e., $\mu_i = g^{-1}(\eta_i)$.

2.1 The Canonical Link

The canonical link sets $\theta_i = \eta_i$ (the canonical parameter equals the linear predictor). It simplifies the score equations and ensures the sufficient statistic $\bX^T\mathbf{y}$ is complete.

3. Estimation: Score, Fisher Information, IRLS

3.1 Score Equations

Theorem 3.1 — Score Equations for a GLM
$$\mathbf{s}(\bbeta) = \frac{\partial\ell}{\partial\bbeta} = \bX^T\bW(\eta)\,\mathbf{d}(\mu)\odot(\mathbf{y}-\boldsymbol\mu) = \mathbf{0},$$

where $\odot$ denotes elementwise product, and the weight for observation $i$ is:

$$w_i = \frac{1}{V(\mu_i)\,[g'(\mu_i)]^2\,a(\phi)}.$$

For the canonical link, $g'(\mu) = 1/V(\mu)$ and the score simplifies to $\bX^T(\mathbf{y}-\boldsymbol\mu)/a(\phi) = \mathbf{0}$.

Derivation via chain rule
Derivation

By the chain rule:

$$\frac{\partial\ell}{\partial\beta_j} = \sum_i \frac{\partial\ell_i}{\partial\theta_i}\frac{\partial\theta_i}{\partial\mu_i}\frac{\partial\mu_i}{\partial\eta_i}\frac{\partial\eta_i}{\partial\beta_j}.$$
  • $\frac{\partial\ell_i}{\partial\theta_i} = (y_i - b'(\theta_i))/a(\phi) = (y_i-\mu_i)/a(\phi)$.
  • $\frac{\partial\theta_i}{\partial\mu_i} = 1/b''(\theta_i) = 1/V(\mu_i)$ (since $\mu = b'(\theta)$ inverted).
  • $\frac{\partial\mu_i}{\partial\eta_i} = 1/g'(\mu_i)$.
  • $\frac{\partial\eta_i}{\partial\beta_j} = x_{ij}$.

Combining: $s_j(\bbeta) = \sum_i \frac{(y_i-\mu_i)x_{ij}}{V(\mu_i)g'(\mu_i)a(\phi)}$. $\square$

3.2 Fisher Information

$$\mathcal{I}(\bbeta) = \bX^T\bW\bX/a(\phi), \quad w_i = \frac{1}{V(\mu_i)[g'(\mu_i)]^2}.$$

3.3 IRLS for GLMs

Algorithm 3.1 — IRLS (General GLM)

At iteration $t$:

  1. Compute $\mu_i^{(t)} = g^{-1}(\eta_i^{(t)})$, weights $w_i^{(t)} = [V(\mu_i^{(t)})(g'(\mu_i^{(t)}))^2]^{-1}$.
  2. Form the adjusted dependent variable: $z_i^{(t)} = \eta_i^{(t)} + (y_i-\mu_i^{(t)})\,g'(\mu_i^{(t)})$.
  3. Update: $\bbeta^{(t+1)} = (\bX^T\bW^{(t)}\bX)^{-1}\bX^T\bW^{(t)}\mathbf{z}^{(t)}$.

This is Newton–Raphson (= Fisher scoring for canonical links), and it is exactly what R's glm() runs.

4. Deviance and Goodness of Fit

4.1 Definition of Deviance

Definition 4.1 — Deviance

The deviance compares the fitted model to the saturated model (one parameter per observation, $\hat\mu_i^{(\text{sat})} = y_i$):

$$D(\mathbf{y};\hat\boldsymbol\mu) = 2\bigl[\ell(\mathbf{y};\mathbf{y}) - \ell(\hat\boldsymbol\mu;\mathbf{y})\bigr] = 2\sum_{i=1}^n d_i,$$

where $d_i = y_i(\tilde\theta_i - \hat\theta_i) - [b(\tilde\theta_i)-b(\hat\theta_i)]$ is the individual deviance contribution, $\tilde\theta_i$ evaluated at $y_i$, $\hat\theta_i$ at $\hat\mu_i$.

FamilyDeviance $D(\mathbf{y};\hat\boldsymbol\mu)$
Normal$\sum(y_i - \hat\mu_i)^2 = \text{RSS}$
Binomial$(m_i,p_i)$$2\sum[y_i\log(y_i/\hat\mu_i) + (m_i-y_i)\log((m_i-y_i)/(m_i-\hat\mu_i))]$
Poisson$2\sum[y_i\log(y_i/\hat\mu_i) - (y_i-\hat\mu_i)]$
Gamma$2\sum[-\log(y_i/\hat\mu_i) + (y_i-\hat\mu_i)/\hat\mu_i]$

4.2 Analysis of Deviance — LRT for GLMs

Theorem 4.1 — Analysis of Deviance

To compare a reduced model $M_0$ ($q$ fewer parameters) against full model $M_1$:

$$G^2 = D_0 - D_1 = \frac{D_0 - D_1}{\phi} \xrightarrow{d} \chi^2_q \quad\text{under }H_0.$$

For the normal family with known $\sigma^2$, this is exact; for other families, it is asymptotic. If $\phi$ is estimated, use $F = G^2/(q\hat\phi)$ with an $F_{q, n-p-1}$ approximation.

6. Overdispersion

In real data, the variance often exceeds what the exponential family predicts. For Poisson, $\Var(Y) = \mu$ by definition, but empirically $\Var(Y) \approx \phi\mu$ with $\phi > 1$.

Definition 6.1 — Overdispersion

The data are overdispersed relative to the nominal model when $\Var(Y_i) > V(\mu_i)a(\phi_{\text{nominal}})$. An informal diagnostic: if the residual deviance $\gg$ its df, overdispersion is likely.

$$\hat\phi = \frac{X^2}{n-p-1} = \frac{\sum(y_i-\hat\mu_i)^2/V(\hat\mu_i)}{n-p-1}.$$

If $\hat\phi \gg 1$, multiply all SEs by $\sqrt{\hat\phi}$ (quasi-likelihood) or switch to Negative Binomial (for counts) / Beta-Binomial (for proportions).

7. Special GLMs in Detail

7.1 Poisson Regression

Definition 7.1 — Poisson Log-Linear Model

$Y_i \sim \text{Poisson}(\mu_i)$ with log link: $\log\mu_i = \bx_i^T\bbeta$, so $\mu_i = e^{\bx_i^T\bbeta}$.

  • Variance function: $V(\mu) = \mu$ (variance $=$ mean).
  • Deviance: $D = 2\sum[y_i\log(y_i/\hat\mu_i)-(y_i-\hat\mu_i)]$.
  • Coefficient interpretation: $e^{\beta_j}$ is the multiplicative factor on the rate for a one-unit increase in $x_j$ — the incidence rate ratio (IRR).

7.2 Offsets in Poisson Regression

When $Y_i$ counts events over a varying exposure (time, population, area), add an offset:

$$\log\mu_i = \log t_i + \bx_i^T\bbeta, \quad \text{i.e., } \mu_i = t_i e^{\bx_i^T\bbeta}.$$

The offset $\log t_i$ is included with a coefficient fixed at 1, not estimated. In R: glm(y ~ x + offset(log(t)), family = poisson).

7.3 Gamma Regression

Definition 7.2 — Gamma GLM

$Y_i \sim \text{Gamma}(\nu, \nu/\mu_i)$ with variance $\Var(Y_i) = \mu_i^2/\nu$ (i.e., $V(\mu) = \mu^2$, constant CV).

  • Canonical link: reciprocal $g(\mu) = 1/\mu$; but the log link is almost always preferred (ensures $\mu_i > 0$ and gives interpretable multiplicative effects).
  • Applications: insurance claims, waiting times, any positive continuous response with variance proportional to mean squared.

The shape parameter $\nu$ (= $1/\phi$ in the exponential family parameterisation) is estimated from the Pearson statistic: $\hat\nu = 1/\hat\phi$.

7.4 Negative Binomial Regression

The Negative Binomial adds a quadratic variance term to handle overdispersed counts:

$$V(\mu) = \mu + \mu^2/r, \quad r > 0 \text{ (size parameter, estimated)}.$$

As $r\to\infty$, NB$\to$Poisson. NB is outside the exponential family (because $r$ is also estimated), so it is fit iteratively — alternating Poisson IRLS for $\bbeta$ given $r$, and moment estimation of $r$. In R: MASS::glm.nb().

9. Complete R Examples with Annotated Output

9.1 Poisson GLM — Hospital Admissions

## ── 1. Simulate hospital admission data ──────────────────────
set.seed(123)
n <- 200
age     <- rnorm(n, 55, 15)
smoker  <- rbinom(n, 1, 0.3)
# log rate = -2 + 0.03*age + 0.7*smoker
lambda  <- exp(-2 + 0.03 * age + 0.7 * smoker)
admits  <- rpois(n, lambda)

dat <- data.frame(admits, age, smoker = factor(smoker))

## ── 2. Fit Poisson GLM ───────────────────────────────────────
fit_pois <- glm(admits ~ age + smoker,
                family = poisson(link = "log"),
                data   = dat)
summary(fit_pois)
# Coefficients:
#              Estimate Std. Error z value Pr(>|z|)
# (Intercept) -1.9631     0.1923  -10.21  < 2e-16 ***
# age          0.0302     0.0031    9.72  < 2e-16 ***
# smoker1      0.6714     0.0861    7.80  6.4e-15 ***
#
# Null deviance: 342.87 on 199 df
# Residual deviance: 193.82 on 197 df
# AIC: 648.84

## ── 3. Incidence Rate Ratios (IRR) ───────────────────────────
IRR_table <- exp(cbind(IRR = coef(fit_pois), confint(fit_pois)))
round(IRR_table, 3)
#                IRR   2.5 %  97.5 %
# (Intercept)  0.141   0.096   0.204
# age          1.031   1.024   1.037   ← each year: 3.1% more admissions
# smoker1      1.957   1.652   2.322   ← smokers: 96% more admissions

## ── 4. LRT: does smoking matter? ─────────────────────────────
fit_no_smoke <- glm(admits ~ age, family = poisson, data = dat)
anova(fit_no_smoke, fit_pois, test = "LRT")
# Analysis of Deviance Table
#                Df Deviance Resid.Df Resid.Dev Pr(>Chi)
# fit_no_smoke                    198    254.96
# fit_pois       1    61.14      197    193.82  5.5e-15 ***

exp(coef(fit_rate)["smoker1"])
# smoker1: 1.952   ← IRR unchanged by offset (as expected)

## ── 1. Simulate insurance claim data ─────────────────────────
set.seed(456)
n <- 300

claims <- rgamma(n, shape = 3, rate = 3 / exp(log_mu))

9.3 Negative Binomial — Overdispersed Counts

## ── 1. Simulate overdispersed count data ─────────────────────
library(MASS)
set.seed(789)
n   <- 250
x   <- rnorm(n)
mu  <- exp(1 + 0.5 * x)
y_nb <- rnegbin(n, mu = mu, theta = 2)   # true NB, theta=2

dat_nb <- data.frame(y = y_nb, x)

## ── 2. Fit Poisson (misspecified) ────────────────────────────
fit_p  <- glm(y ~ x, family = poisson, data = dat_nb)
phi_check <- sum(residuals(fit_p,"pearson")^2) / fit_p$df.residual
cat("Poisson dispersion:", round(phi_check, 2))
# Poisson dispersion: 1.46    ← clearly > 1; overdispersed

## ── 3. Quasi-Poisson (corrects SEs but not distribution) ─────
fit_qp <- glm(y ~ x, family = quasipoisson, data = dat_nb)
summary(fit_qp)$coefficients
#             Estimate Std. Error t value Pr(>|t|)
# (Intercept)    1.013     0.0710   14.27  < 2e-16 ***
# x              0.495     0.0591    8.38  2.8e-15 ***
# (SEs inflated by sqrt(1.46) compared to Poisson)

## ── 4. Negative Binomial (correct model) ─────────────────────
fit_nb <- glm.nb(y ~ x, data = dat_nb)
summary(fit_nb)
# Coefficients:
#             Estimate Std. Error z value Pr(>|z|)
# (Intercept)   1.0147     0.0690   14.71  < 2e-16 ***
# x             0.4987     0.0575    8.67  < 2e-16 ***
# Theta: 1.994 (SE: 0.250)   ← recovers true theta ≈ 2

## ── 5. LRT: Poisson vs NB ─────────────────────────────────────
# 2*(logLik(NB) - logLik(Poisson)) ~ chi-sq(1)
lrt_nb <- 2*(as.numeric(logLik(fit_nb)) - as.numeric(logLik(fit_p)))
cat("LRT statistic:", round(lrt_nb, 2), "\n")
cat("p-value:", pchisq(lrt_nb, 1, lower.tail = FALSE)/2, "\n")
# LRT statistic: 56.8
# p-value: 2.3e-14   ← strong evidence for overdispersion

## ── 6. Summary comparison ────────────────────────────────────
AIC(fit_p, fit_nb)
#        df      AIC
# fit_p   2  1182.4
# fit_nb  3  1127.5    ← NB substantially better

Key Takeaways

Summary

  • The exponential family unifies all GLM distributions via $f = \exp\{[y\theta - b(\theta)]/a(\phi) + c(y,\phi)\}$; mean $= b'(\theta)$, variance $= a(\phi)b''(\theta)$.
  • A GLM has three components: random (family), systematic (linear predictor $\bx^T\bbeta$), and link ($g(\mu) = \eta$).
  • All GLMs are estimated by IRLS — the same algorithm as logistic regression, but with family-specific weights $w_i = 1/[V(\mu_i)(g'(\mu_i))^2]$.
  • Inference uses the deviance; the LRT $G^2 = D_0 - D_1 \sim \chi^2_q$ is the GLM analogue of the $F$-test.
  • Check overdispersion (Pearson $\hat\phi \gg 1$); if present, switch to quasi-likelihood (corrects SEs only) or a more flexible family (NB, Beta-Binomial).
  • Choose the family by the response type and variance function; choose the link by the scale on which linearity makes subject-matter sense.
