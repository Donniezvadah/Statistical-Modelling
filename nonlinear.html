Nonlinear Models

When a straight line — or even a polynomial — is not enough

3.1 Motivation: Why Go Nonlinear?

In all the models we have encountered so far, the expected response is a linear function of parameters: $\mathbb{E}[Y] = \mathbf{x}^\top \boldsymbol{\beta}$, possibly after a link transformation. While flexible (polynomials, splines, interactions), this linearity imposes a structure that may not reflect the underlying science.

Consider biological growth, pharmacokinetics (drug absorption and elimination), enzyme kinetics, or chemical reactions. These processes are governed by differential equations whose solutions are inherently nonlinear — exponential saturation, sigmoidal growth, double-exponential decline. Forcing these into a linear framework requires many parameters, produces physically uninterpretable coefficients, and often gives poor fits.

💡 Physical vs. empirical parameters

A key advantage of nonlinear models is that their parameters often carry direct physical meaning:

  • Physical (mechanistic) parameters: directly interpretable quantities such as asymptotic level, half-life, rate constant, or carrying capacity.
  • Tuning (empirical) parameters: improve model fit but lack direct scientific interpretation (e.g., a 7th-degree polynomial coefficient).

Using physical parameters makes models more interpretable, reduces the number of parameters needed, and allows prior information to be incorporated naturally.

3.2 Nonlinear Regression with Fixed Effects

📖 Definition 3.1 — Nonlinear Regression Model

Given responses $y_1, \ldots, y_n$ and covariates $x_1, \ldots, x_n$, the nonlinear regression model specifies:

$$Y_i = \eta(x_i, \boldsymbol{\beta}) + \varepsilon_i, \qquad \varepsilon_i \overset{\mathrm{i.i.d.}}{\sim} \mathrm{N}(0, \sigma^2)$$

where $\eta(\cdot, \boldsymbol{\beta})$ is a known nonlinear function of $\boldsymbol{\beta} \in \mathbb{R}^p$. The function $\eta$ encodes the mathematical structure of the mean response.

Estimation: Nonlinear Least Squares

Under the normality assumption, maximising the log-likelihood is equivalent to nonlinear least squares (NLS) — minimising:

$$S(\boldsymbol{\beta}) = \sum_{i=1}^n (y_i - \eta(x_i, \boldsymbol{\beta}))^2$$

Unlike ordinary least squares, there is no closed-form solution. We must use iterative optimisation. The most widely used method is Gauss–Newton.

The Gauss–Newton Algorithm — Step by Step

The Gauss–Newton algorithm linearises the model at the current parameter estimate and takes an ordinary least squares step.

  1. Initialise: Choose a starting value $\boldsymbol{\beta}^{(0)}$ (good starting values are crucial — see below).
  2. Linearise: Compute the residuals $r_i^{(t)} = y_i - \eta(x_i, \boldsymbol{\beta}^{(t)})$ and the Jacobian matrix $$\mathbf{J}^{(t)}, \quad J_{ij}^{(t)} = \frac{\partial \eta(x_i, \boldsymbol{\beta})}{\partial \beta_j}\Bigg|_{\boldsymbol{\beta} = \boldsymbol{\beta}^{(t)}}$$ This is the $n \times p$ matrix of partial derivatives.
  3. Update: Solve the linearised OLS problem: $$\boldsymbol{\beta}^{(t+1)} = \boldsymbol{\beta}^{(t)} + (\mathbf{J}^{(t)\top} \mathbf{J}^{(t)})^{-1} \mathbf{J}^{(t)\top} \mathbf{r}^{(t)}$$
  4. Check convergence: Stop when $\|S(\boldsymbol{\beta}^{(t+1)}) - S(\boldsymbol{\beta}^{(t)})\| < \epsilon$ or $\|\boldsymbol{\beta}^{(t+1)} - \boldsymbol{\beta}^{(t)}\| < \delta$.
💡 Why good starting values matter so much

Unlike linear regression, $S(\boldsymbol{\beta})$ may have many local minima. Gauss–Newton will converge to whichever local minimum is nearest to the starting point. Strategies for good starts:

  • Use physical knowledge of parameter ranges
  • Plot $\eta(x, \boldsymbol{\beta})$ for candidate values and match visually
  • Use getInitial() in R, which provides self-starting functions for common models
  • Grid search over a coarse parameter space first

Inference for NLS

Let $\hat{\boldsymbol{\beta}}$ be the NLS estimator and $\hat{\mathbf{J}} = \mathbf{J}(\hat{\boldsymbol{\beta}})$ the Jacobian at the solution. Under standard regularity conditions:

📐 Theorem 3.1 — Asymptotic Distribution of NLS Estimator

As $n \to \infty$:

$$\sqrt{n}(\hat{\boldsymbol{\beta}} - \boldsymbol{\beta}_0) \overset{d}{\to} \mathrm{N}\!\left(\mathbf{0},\, \sigma^2 \left(\mathbf{J}^\top \mathbf{J}/n\right)^{-1}\right)$$

In practice, estimate $\sigma^2$ by $\hat{\sigma}^2 = S(\hat{\boldsymbol{\beta}})/(n-p)$ and the covariance by:

$$\widehat{\mathrm{Var}}(\hat{\boldsymbol{\beta}}) = \hat{\sigma}^2 (\hat{\mathbf{J}}^\top \hat{\mathbf{J}})^{-1}$$
Proof sketch of Theorem 3.1

The NLS estimator satisfies $\hat{\mathbf{J}}^\top (\mathbf{y} - \eta(\hat{\boldsymbol{\beta}})) = \mathbf{0}$ (first-order conditions). Taylor-expand around the true $\boldsymbol{\beta}_0$:

$$\mathbf{0} \approx \mathbf{J}_0^\top (\mathbf{y} - \eta(\boldsymbol{\beta}_0)) - \mathbf{J}_0^\top \mathbf{J}_0 (\hat{\boldsymbol{\beta}} - \boldsymbol{\beta}_0)$$

Rearranging:

$$\hat{\boldsymbol{\beta}} - \boldsymbol{\beta}_0 \approx (\mathbf{J}_0^\top \mathbf{J}_0)^{-1} \mathbf{J}_0^\top \boldsymbol{\varepsilon}$$

Since $\boldsymbol{\varepsilon} \sim \mathrm{N}(\mathbf{0}, \sigma^2\mathbf{I})$, this gives $\hat{\boldsymbol{\beta}} - \boldsymbol{\beta}_0 \sim \mathrm{N}(\mathbf{0}, \sigma^2(\mathbf{J}_0^\top\mathbf{J}_0)^{-1})$ exactly in the linearised model, and approximately in general.

💼 Example 3.1 — Calcium Uptake in Cells

Calcium uptake (nmoles/mg) was measured at $n = 27$ time points (minutes) for individual cells. The underlying mechanism: calcium flows into the cell at a rate proportional to the remaining capacity, giving the ODE $\mathrm{d}U/\mathrm{d}t = \beta_0/\beta_1 \cdot e^{-t/\beta_1}$, with solution $U(t) = \beta_0(1 - e^{-t/\beta_1})$. So $\beta_0$ = asymptotic uptake, $\beta_1$ = time constant.

R
library(nlme)
data(calcium)   # 27 observations: time (x), uptake (y)

# Fit nonlinear model using nls()
fit_nls <- nls(uptake ~ beta0 * (1 - exp(-time / beta1)),
               data = calcium,
               start = list(beta0 = 5, beta1 = 2))

summary(fit_nls)
# Confidence intervals
confint(fit_nls)

# Compare with linear and quadratic models
fit_lin  <- lm(uptake ~ time,               data = calcium)
fit_quad <- lm(uptake ~ time + I(time^2),   data = calcium)

# AIC comparison
AIC(fit_lin, fit_quad, fit_nls)
# Nonlinear model wins: fewer params, better physical interpretation

# Plot fitted curve
curve_data <- data.frame(time = seq(0, max(calcium$time), length.out = 200))
curve_data$fitted <- predict(fit_nls, newdata = curve_data)

plot(calcium$time, calcium$uptake, pch = 16,
     xlab = "Time (min)", ylab = "Calcium Uptake (nmol/mg)",
     main = "Calcium uptake: NLS fit")
lines(curve_data$time, curve_data$fitted, col = "steelblue", lwd = 2)

The nonlinear model achieves a better AIC than both the linear and quadratic alternatives while using only 2 parameters instead of 3. The parameter $\hat{\beta}_1$ (the time constant) has direct physical meaning: it is the time at which approximately 63% of the asymptotic uptake is achieved.

3.3 Nonlinear Mixed Effects Models

In many applications, several individuals (subjects, animals, plants) are each measured repeatedly over time or over a range of covariate values. A single nonlinear curve fitted to all observations will miss the between-subject variability. The solution: allow the parameters $\boldsymbol{\beta}$ to vary randomly across individuals.

📖 Definition 3.2 — Nonlinear Mixed Effects Model (NLME)

For observations $Y_{ij}$ in cluster $i$ ($i = 1,\ldots,m$; $j = 1,\ldots,n_i$):

$$Y_{ij} = \eta(\boldsymbol{\beta} + \mathbf{b}_i,\, x_{ij}) + \varepsilon_{ij}$$

where:

  • $\eta(\cdot, x)$ is a nonlinear function of both parameters and covariate $x$
  • $\boldsymbol{\beta}$ — population-level (fixed) parameters
  • $\mathbf{b}_i \sim \mathrm{N}(\mathbf{0}, \boldsymbol{\Sigma}_b)$ — subject-specific random deviations
  • $\varepsilon_{ij} \sim \mathrm{N}(0, \sigma^2)$ — within-subject noise, independent of $\mathbf{b}_i$

Subject $i$ has an individual parameter vector $\boldsymbol{\beta}_i = \boldsymbol{\beta} + \mathbf{b}_i$.

Partial Random Effects

It is often scientifically justified to allow only a subset of parameters to vary across subjects. For instance, in pharmacokinetics the absorption rate may be relatively constant across patients while clearance varies substantially. This is accommodated by a selection matrix $\mathbf{A}$ ($p \times q$, $q \leq p$):

$$Y_{ij} = \eta(\boldsymbol{\beta} + \mathbf{A}\mathbf{b}_i,\, x_{ij}) + \varepsilon_{ij}, \qquad \mathbf{b}_i \sim \mathrm{N}(\mathbf{0}, \boldsymbol{\Sigma}_b)$$

where $\mathbf{A}$ has entries $0$ or $1$ selecting which parameters have random variation.

Log-Scale Parameterisation

Many physical parameters are constrained to be positive (rates, concentrations, half-lives). To ensure positivity and improve numerical behaviour, work on the log scale:

$$\gamma_k = \log \beta_k, \quad \boldsymbol{\gamma} = \log \boldsymbol{\beta}$$

Then $\boldsymbol{\gamma}_i = \boldsymbol{\gamma} + \mathbf{b}_i$ with $\mathbf{b}_i \sim \mathrm{N}(\mathbf{0}, \boldsymbol{\Sigma}_b)$ implies $\boldsymbol{\beta}_i = \exp(\boldsymbol{\gamma}_i)$ follows a log-normal distribution. This is standard in pharmacokinetic modelling.

Estimation

The marginal likelihood for NLMEs requires integrating out random effects: $$\ell(\boldsymbol{\gamma}, \boldsymbol{\Sigma}_b, \sigma^2) = \sum_{i=1}^m \log \int f(\mathbf{y}_i \mid \mathbf{b}_i; \boldsymbol{\gamma}, \sigma^2) \, \phi(\mathbf{b}_i; \boldsymbol{\Sigma}_b) \, d\mathbf{b}_i$$

Since $Y_{ij} \mid \mathbf{b}_i$ is normal (not just exponential family), a Laplace approximation works very well. The nlme and nlmer functions in R implement this via adaptive Gaussian quadrature or Laplace.

💼 Example 3.2 — Theophylline Pharmacokinetics

Theophylline is a drug used to treat asthma. After oral administration of dose $D$, plasma concentration follows a one-compartment model with first-order absorption and elimination:

$$\eta(\boldsymbol{\beta}, D, t) = \frac{D \beta_1 \beta_2}{\beta_3(\beta_2 - \beta_1)} \left(e^{-\beta_1 t} - e^{-\beta_2 t}\right)$$

where:

  • $\beta_1$ — elimination rate constant (h$^{-1}$)
  • $\beta_2$ — absorption rate constant (h$^{-1}$)
  • $\beta_3$ — apparent clearance (L/h)

Data: 12 subjects, each measured at 11 time points. Each subject receives dose $D = 320$mg.

R
library(nlme)
library(lme4)

data(Theoph)  # Built into R

# ── Step 1: Fit fixed-effects NLS (ignores clustering) ──────────────
pk_model <- function(D, t, b1, b2, b3) {
  D * b1 * b2 / (b3 * (b2 - b1)) * (exp(-b1 * t) - exp(-b2 * t))
}

fit_nls <- nls(conc ~ pk_model(dose, Time, exp(g1), exp(g2), exp(g3)),
               data   = Theoph,
               start  = list(g1 = log(0.08), g2 = log(1.5), g3 = log(0.04)))

# ── Step 2: Diagnose clustering ──────────────────────────────────────
Theoph$resid_nls <- residuals(fit_nls)
# Plot residuals by subject — should see consistent patterns
library(ggplot2)
ggplot(Theoph, aes(x = Time, y = resid_nls, group = Subject, colour = Subject)) +
  geom_line() + geom_point() +
  labs(title = "NLS residuals: between-subject patterns indicate mixed model is needed")

# ── Step 3: Fit NLME — all three parameters vary by subject ─────────
pk_nlme <- nlme(conc ~ pk_model(dose, Time, exp(g1+b1), exp(g2+b2), exp(g3+b3)),
                data   = groupedData(conc ~ Time | Subject, data = Theoph),
                fixed  = g1 + g2 + g3 ~ 1,
                random = g1 + g2 + g3 ~ 1,
                start  = fixef(fit_nls))

# ── Step 4: Simpler NLME — only elimination and clearance vary ───────
pk_nlme_simple <- nlme(conc ~ pk_model(dose, Time, exp(g1+b1), exp(g2), exp(g3+b3)),
                       data   = groupedData(conc ~ Time | Subject, data = Theoph),
                       fixed  = g1 + g2 + g3 ~ 1,
                       random = g1 + g3 ~ 1,
                       start  = fixef(fit_nls))

# Compare by BIC (use ML)
AIC(pk_nlme, pk_nlme_simple)
BIC(pk_nlme, pk_nlme_simple)

BIC typically favours the simpler model where only elimination rate ($\beta_1$) and clearance ($\beta_3$) have random effects. The absorption rate $\beta_2$ does not vary significantly between subjects — a biologically sensible conclusion.

Individual vs. Population Predictions

An NLME model gives two types of predictions:

Prediction typeParameters usedFormulaUse case
Population prediction Fixed effects only $\hat{Y}_{ij}^{\mathrm{pop}} = \eta(\hat{\boldsymbol{\gamma}}, x_{ij})$ Predict a new, unobserved subject
Individual prediction Fixed + random effects $\hat{Y}_{ij}^{\mathrm{ind}} = \eta(\hat{\boldsymbol{\gamma}} + \hat{\mathbf{b}}_i, x_{ij})$ Predict for an observed subject with data

3.4 Generalised Nonlinear Mixed Effects Models (GNLMMs)

Combining everything so far, we arrive at the most general model in this hierarchy: non-normal responses (exponential family) with a nonlinear mean function and random effects.

📖 Definition 3.3 — Generalised Nonlinear Mixed Effects Model

For cluster $i$, observation $j$, with random effects $\mathbf{b}_i \sim \mathrm{N}(\mathbf{0}, \boldsymbol{\Sigma}_b)$:

$$Y_{ij} \mid \mathbf{b}_i \sim \mathrm{EF}(\mu_{ij}, \phi), \qquad g(\mu_{ij}) = \eta(\boldsymbol{\beta} + \mathbf{A}\mathbf{b}_i,\, x_{ij})$$

where $g$ is a link function, EF is exponential family, and $\eta$ is a known nonlinear function (not necessarily a linear predictor).

Model Hierarchy

The GNLMM is the most general class we study. All previously encountered models are special cases, obtained by appropriate restrictions:

📐 The complete model hierarchy
GNLMM:  Y|b ~ EF(μ),  g(μ) = η(β+Ab, x)
  │
  ├─ Set b=0:          Generalised Nonlinear Model (GNM)
  │   ├─ Set g=identity:  Nonlinear Regression (NLS)
  │   └─ Set η(β,x)=x'β:  GLM
  │
  ├─ Set η(β,x)=x'β+z'b:  GLMM
  │   └─ Set EF=Normal:    LMM
  │       └─ Set b=0:       Linear Model
  │
  └─ Set EF=Normal:    NLME

Practical Challenges

GNLMMs present several difficulties that are active areas of research:

  1. No closed-form likelihood. Integration over $\mathbf{b}_i$ is required. Gauss–Hermite quadrature becomes expensive for $\dim(\mathbf{b}_i) > 2$–$3$. Laplace approximation may be inaccurate when the random effect distribution is non-Gaussian or heavy-tailed.
  2. Non-convex optimisation. Unlike GLMs, the nonlinear mean function can create a non-concave log-likelihood with multiple local maxima. Good starting values and multiple restarts are essential.
  3. Numerically defined $\eta$. When $\eta$ is defined implicitly via an ODE system (common in pharmacokinetics), each gradient evaluation requires solving the ODE — very expensive.
  4. Identifiability. Some parameters may not be identifiable from the data alone (e.g., if $\beta_1$ and $\beta_2$ enter $\eta$ only through their difference).
💼 Example 3.3 — GNLMM for Binary Dose-Response

In a drug trial, binary response (success/failure) varies with dose $x_{ij}$. A 4-parameter logistic (4PL) model is more biologically realistic than a simple logistic regression:

$$P(Y_{ij} = 1 \mid b_i) = \frac{\beta_0 + b_i}{\left(1 + (x_{ij}/\beta_2)^{\beta_3}\right)} \cdot \beta_1$$
R
library(lme4)

# Simulate 4PL dose-response data
set.seed(2024)
m <- 20; ni <- 15
dose <- rep(10^seq(-2, 2, length.out = ni), m)
subject <- rep(1:m, each = ni)

b <- rnorm(m, 0, 0.3)
beta0 <- 0.05; beta1 <- 0.95; beta2 <- 1; beta3 <- 2

mu <- beta0 + (beta1 - beta0) /
  (1 + (dose / beta2)^beta3) + b[subject]
mu <- pmin(pmax(mu, 0.001), 0.999)
y  <- rbinom(m * ni, 1, mu)
df <- data.frame(y, dose, subject = factor(subject))

# Fit GLMM with logistic link (approximation to 4PL)
fit_glmm <- glmer(y ~ log(dose) + I(log(dose)^2) + (1|subject),
                  data = df, family = binomial, nAGQ = 15)
summary(fit_glmm)

3.5 Model Selection for Nonlinear Models

The same principles from Chapter 1 apply to nonlinear models, but with some important subtleties.

AIC and BIC for NLS

For a normal-errors nonlinear model with $p$ parameters and $\hat{\sigma}^2 = S(\hat{\boldsymbol{\beta}})/n$:

$$\ell(\hat{\boldsymbol{\beta}}, \hat{\sigma}^2) = -\frac{n}{2}\log(2\pi\hat{\sigma}^2) - \frac{n}{2}$$ $$\mathrm{AIC} = 2(p+1) - 2\ell(\hat{\boldsymbol{\beta}}, \hat{\sigma}^2) = n\log(\hat{\sigma}^2) + 2(p+1) + \mathrm{const}$$

The $+1$ accounts for the estimated $\sigma^2$. In R, AIC(fit_nls) handles this automatically.

Profile Likelihood and Confidence Regions

Standard Wald intervals (± 1.96 SE) can be badly asymmetric for nonlinear parameters. Profile likelihood intervals are more reliable:

$$\mathrm{CI}_{1-\alpha}(\beta_k) = \left\{\beta_k : 2[\ell(\hat{\boldsymbol{\beta}}) - \ell_p(\beta_k)] \leq \chi^2_{1, 1-\alpha}\right\}$$

where $\ell_p(\beta_k) = \max_{\boldsymbol{\beta}_{-k}} \ell(\boldsymbol{\beta})$ is the profile log-likelihood. In R: confint(fit_nls) computes profile likelihood intervals.

Chapter Summary

Key Points from Chapter 3

  • Nonlinear regression $Y_i = \eta(x_i, \boldsymbol{\beta}) + \varepsilon_i$ is appropriate when the mean function is known to be nonlinear from scientific principles.
  • Estimation uses nonlinear least squares (Gauss–Newton), which requires good starting values and may converge to local minima.
  • Asymptotic inference is based on the Jacobian matrix $\mathbf{J}$ evaluated at $\hat{\boldsymbol{\beta}}$; profile likelihood intervals are preferred over Wald intervals for potentially skewed sampling distributions.
  • NLMEs extend nonlinear regression to clustered data by adding subject-specific random deviations $\mathbf{b}_i \sim \mathrm{N}(\mathbf{0}, \boldsymbol{\Sigma}_b)$.
  • Log-scale parameterisation ensures positivity and stabilises the optimisation.
  • The GNLMM is the most general model in the hierarchy, nesting GLMs, LMMs, NLMEs, and GLMMs as special cases.
