Logistic Regression

When the outcome is binary — event/no-event, diseased/healthy, pass/fail — linear regression gives nonsensical predictions. Logistic regression is the canonical fix: it models the probability of the event through the logit link, respects the $[0,1]$ constraint, and is estimated by maximum likelihood via the elegant IRLS algorithm. This chapter covers every step from first principles, including full derivations, interpretation of odds ratios, ROC analysis, and extensions to multinomial and ordinal outcomes.

1. Why Not Linear Regression for Binary Outcomes?

Let $Y_i \in \{0,1\}$. The linear probability model $\E[Y_i\mid\bx_i]=\bx_i^T\bbeta$ suffers from:

  • Out-of-range predictions: $\hat p_i$ can fall outside $[0,1]$.
  • Heteroscedasticity: $\Var(Y_i)=p_i(1-p_i)$ varies with $i$, violating OLS assumptions.
  • Non-linearity: for extreme $\bx$, the true probability curve is S-shaped, not straight.

The solution is to apply a monotone link function $g$ so that $g(p_i) = \bx_i^T\bbeta$, mapping $p_i\in(0,1)$ to the whole real line.

2. The Logistic Regression Model

Definition 2.1 — Logistic Regression

Let $Y_i \mid \bx_i \sim \mathrm{Bernoulli}(p_i)$ independently. The logistic model specifies:

$$\mathrm{logit}(p_i) := \log\frac{p_i}{1-p_i} = \eta_i = \bx_i^T\bbeta,$$

equivalently, inverting the logit (the logistic/sigmoid function):

$$p_i = \sigma(\eta_i) := \frac{e^{\eta_i}}{1+e^{\eta_i}} = \frac{1}{1+e^{-\eta_i}} \in (0,1).$$

Logit and probit give virtually identical fitted probabilities in practice; the logit is preferred for its interpretability (log-odds).

3. Maximum Likelihood Estimation

3.1 Log-Likelihood

The log-likelihood for $n$ independent Bernoulli observations:

$$\ell(\bbeta) = \sum_{i=1}^n \bigl[y_i\log p_i + (1-y_i)\log(1-p_i)\bigr] = \sum_{i=1}^n \bigl[y_i\eta_i - \log(1+e^{\eta_i})\bigr].$$
Theorem 3.1 — Concavity of the Log-Likelihood

$\ell(\bbeta)$ is strictly concave in $\bbeta$ whenever $\bX$ has full column rank. This guarantees a unique global maximum.

Proof of concavity
Proof

The Hessian is $\nabla^2\ell = -\bX^T\bW\bX$ where $\bW = \mathrm{diag}(p_1(1-p_1),\ldots,p_n(1-p_n))$, with all diagonal entries in $(0,\tfrac{1}{4}]$. Since $\bX$ has full column rank, for any $\mathbf{v}\neq\mathbf{0}$: $\mathbf{v}^T\bX^T\bW\bX\mathbf{v} = \|\bW^{1/2}\bX\mathbf{v}\|^2 > 0$ (as $\bW\succ 0$ and $\bX\mathbf{v}\neq\mathbf{0}$). So $\nabla^2\ell \prec 0$ and $\ell$ is strictly concave. $\square$

3.2 Score Equations

Theorem 3.2 — Score Equations
$$\mathbf{s}(\bbeta) := \nabla_\bbeta\ell(\bbeta) = \sum_{i=1}^n(y_i - p_i)\bx_i = \bX^T(\mathbf{y}-\mathbf{p}) = \mathbf{0}.$$

There is no closed-form solution because $\mathbf{p}$ depends non-linearly on $\bbeta$.

Derivation of the score
Derivation

$\frac{\partial\ell}{\partial\beta_j} = \sum_i\left[y_i\frac{\partial\eta_i}{\partial\beta_j} - \frac{e^{\eta_i}}{1+e^{\eta_i}}\frac{\partial\eta_i}{\partial\beta_j}\right] = \sum_i x_{ij}(y_i - p_i),$

since $\frac{\partial\eta_i}{\partial\beta_j} = x_{ij}$ and $p_i = \sigma(\eta_i)$. $\square$

3.3 Fisher Information Matrix

$$\mathcal{I}(\bbeta) = -\E[\nabla^2\ell] = \bX^T\bW\bX, \quad \bW = \mathrm{diag}\bigl(p_1(1-p_1),\ldots,p_n(1-p_n)\bigr).$$

Each weight $w_i = p_i(1-p_i)$ is the Bernoulli variance — observations near $p=0.5$ are most informative.

3.4 IRLS Algorithm

Newton–Raphson applied to the score equations becomes a sequence of weighted least squares problems.

Algorithm 3.1 — Iteratively Reweighted Least Squares (IRLS)

Initialise $\bbeta^{(0)}$ (e.g. all zeros). At iteration $t$:

  1. Compute $\eta_i^{(t)} = \bx_i^T\bbeta^{(t)}$; $\;p_i^{(t)} = \sigma(\eta_i^{(t)})$; $\;w_i^{(t)} = p_i^{(t)}(1-p_i^{(t)})$.
  2. Form the working response: $$z_i^{(t)} = \eta_i^{(t)} + \frac{y_i - p_i^{(t)}}{w_i^{(t)}}.$$
  3. Update by weighted OLS: $$\bbeta^{(t+1)} = \bigl(\bX^T\bW^{(t)}\bX\bigr)^{-1}\bX^T\bW^{(t)}\mathbf{z}^{(t)}.$$
  4. Repeat until convergence: $\|\bbeta^{(t+1)}-\bbeta^{(t)}\|_\infty < 10^{-8}$.

This is exactly the Newton–Raphson update $\bbeta^{(t+1)} = \bbeta^{(t)} + [\mathcal{I}(\bbeta^{(t)})]^{-1}\mathbf{s}(\bbeta^{(t)})$ rewritten as a WLS problem.

Derivation — from Newton–Raphson to WLS
Derivation

Newton–Raphson: $\bbeta^{(t+1)} = \bbeta^{(t)} + (\bX^T\bW^{(t)}\bX)^{-1}\bX^T(\mathbf{y}-\mathbf{p}^{(t)})$.

Define $\mathbf{z}^{(t)} = \boldsymbol\eta^{(t)} + (\bW^{(t)})^{-1}(\mathbf{y}-\mathbf{p}^{(t)})$. Then:

$$\bX^T\bW^{(t)}\mathbf{z}^{(t)} = \bX^T\bW^{(t)}\boldsymbol\eta^{(t)} + \bX^T(\mathbf{y}-\mathbf{p}^{(t)}) = \bX^T\bW^{(t)}\bX\bbeta^{(t)} + \bX^T(\mathbf{y}-\mathbf{p}^{(t)}).$$

So $(\bX^T\bW^{(t)}\bX)^{-1}\bX^T\bW^{(t)}\mathbf{z}^{(t)} = \bbeta^{(t)} + (\bX^T\bW^{(t)}\bX)^{-1}\bX^T(\mathbf{y}-\mathbf{p}^{(t)}) = \bbeta^{(t+1)}$. $\square$

4. Inference

4.1 Wald Tests and CIs

By asymptotic normality of the MLE:

$$\hat\bbeta \xrightarrow{d} N\!\bigl(\bbeta,\;(\bX^T\hat\bW\bX)^{-1}\bigr).$$

Wald statistic for $H_0\colon\beta_j=0$: $\;z_j = \hat\beta_j/\mathrm{SE}(\hat\beta_j) \to N(0,1)$ under $H_0$.

Hauck–Donner effect

Wald tests can give anti-conservative $p$-values (too large) when $|\hat\beta_j|$ is very large — the estimated SE explodes. Always prefer the Likelihood Ratio Test for logistic regression.

4.2 Likelihood Ratio Test (LRT)

Theorem 4.1 — LRT

Comparing a reduced model (with $q$ fewer parameters) to the full model:

$$G^2 = 2\bigl[\ell(\hat\bbeta_\text{full}) - \ell(\hat\bbeta_\text{red})\bigr] = D_\text{red} - D_\text{full} \xrightarrow{d}\chi^2_q \;\text{ under }H_0,$$

where the deviance $D = -2\ell(\hat\bbeta)$. Reject $H_0$ if $G^2 > \chi^2_{q,1-\alpha}$.

Deviance = residual log-likelihood ratio

$D = -2\ell(\hat\bbeta)$ measures how far the fitted model is from a saturated model (one parameter per observation). A smaller deviance means better fit. The null deviance $D_0 = -2\ell(\hat\bbeta_0)$ (intercept-only) is the baseline. The explained deviance reduction $D_0 - D$ plays the same role as SSR in linear regression.

4.3 Profile Likelihood Confidence Intervals

Profile likelihood CIs are more accurate than Wald CIs for logistic regression, especially with moderate $n$. For $\beta_j$:

$$\text{CI}_{1-\alpha}(\beta_j) = \left\{\beta_j : -2[\ell(\hat\bbeta_{\beta_j}) - \ell(\hat\bbeta)] \leq \chi^2_{1,1-\alpha}\right\},$$

where $\hat\bbeta_{\beta_j}$ maximises the likelihood with $\beta_j$ fixed. In R: confint(fit) uses profiling by default.

5. Interpreting Coefficients

5.1 Log-Odds and Odds Ratios

Definition 5.1 — Odds Ratio

A one-unit increase in $x_j$ (others held fixed) changes the log-odds by $\beta_j$:

$$\log\frac{p'}{1-p'} - \log\frac{p}{1-p} = \beta_j.$$

On the odds scale: $\mathrm{OR}_j = e^{\beta_j}$. Confidence interval: $\exp(\hat\beta_j \pm 1.96\,\mathrm{SE}(\hat\beta_j))$.

  • OR $> 1$: $x_j$ increases odds of event.
  • OR $= 1$ ($\beta_j = 0$): no association.
  • OR $< 1$: $x_j$ decreases odds of event.

5.2 Marginal Effects on Probability

The effect of $x_j$ on probability $p_i$ depends on all other covariates:

$$\frac{\partial p_i}{\partial x_j} = \beta_j \cdot p_i(1-p_i).$$

This is maximised at $p_i = 0.5$ (where $p(1-p) = 0.25$) and approaches 0 as $p\to 0$ or $p\to 1$. The Average Marginal Effect (AME):

$$\widehat{\mathrm{AME}}_j = \frac{1}{n}\sum_{i=1}^n\hat\beta_j\hat p_i(1-\hat p_i).$$
Odds ratio ≠ risk ratio (except for rare outcomes)

For common outcomes ($p > 10\%$), ORs overstate the effect size relative to the risk ratio (RR = $p_1/p_0$). Always report both, or explicitly note the outcome prevalence.

7.2 ROC Curve and AUC

Definition 7.1 — ROC and AUC

The ROC curve plots Sensitivity (TPR) against $1-$Specificity (FPR) as the classification threshold $c$ varies from 1 to 0.

The Area Under the Curve (AUC / c-statistic):

$$\mathrm{AUC} = \Pr(\hat p_i > \hat p_j \mid Y_i=1, Y_j=0) \in [0.5, 1].$$

AUC = 0.5: random guessing. AUC = 1: perfect separation. AUC $\geq 0.8$: generally considered good discrimination.

8. Extensions

8.1 Multinomial Logistic Regression

For $Y_i \in \{1,\ldots,K\}$ (unordered), using category $K$ as reference:

$$\log\frac{\Pr(Y_i=k\mid\bx_i)}{\Pr(Y_i=K\mid\bx_i)} = \bx_i^T\bbeta_k, \quad k=1,\ldots,K-1.$$ $$\Pr(Y_i=k\mid\bx_i) = \frac{e^{\bx_i^T\bbeta_k}}{1+\sum_{j=1}^{K-1}e^{\bx_i^T\bbeta_j}}, \quad \Pr(Y_i=K\mid\bx_i) = \frac{1}{1+\sum_{j=1}^{K-1}e^{\bx_i^T\bbeta_j}}.$$

Fit in R: nnet::multinom(). The $(K-1)\times(p+1)$ parameter matrix is estimated jointly by MLE.

8.2 Proportional Odds Model (Ordinal Outcomes)

For ordered $Y_i\in\{1,\ldots,K\}$, the cumulative logit model:

$$\log\frac{\Pr(Y_i \leq k \mid\bx_i)}{\Pr(Y_i > k \mid\bx_i)} = \alpha_k - \bx_i^T\bbeta, \quad k=1,\ldots,K-1,$$

with ordered intercepts $\alpha_1 < \cdots < \alpha_{K-1}$ and a single shared $\bbeta$ (proportional odds assumption). The OR for $\bbeta_j$ is constant across all $K-1$ cutpoints. Fit in R: MASS::polr().

9. Complete Separation and Rare Events

Complete (and Quasi-complete) Separation

If a linear combination of $\bX$ perfectly predicts $Y$, the MLE does not exist: $\|\hat\bbeta\|\to\infty$. Symptoms: enormous SEs, fitted probabilities numerically 0 or 1 warning in R.

Remedy: Firth's penalised likelihood adds $-\frac{1}{2}\log|\mathcal{I}(\bbeta)|$ (Jeffreys prior) to $\ell(\bbeta)$, yielding a finite estimator. Use logistf::logistf().

Key Takeaways

Summary

  • Logistic regression models $\Pr(Y=1\mid\bx)$ through the logit link, guaranteeing predictions in $(0,1)$.
  • The log-likelihood is strictly concave; the unique MLE is found by IRLS, which is Newton–Raphson rewritten as weighted least squares.
  • Coefficients are log-odds ratios; exponentiate to get ORs. Marginal effects on probability depend on the baseline probability.
  • Always use the LRT over the Wald test; avoid Wald CIs when $|\hat\beta|$ is large.
  • Use profile likelihood CIs (confint(fit)) rather than Wald CIs.
  • AUC is threshold-independent and the best single summary of discriminative ability.
  • Firth's penalised likelihood resolves complete separation.
